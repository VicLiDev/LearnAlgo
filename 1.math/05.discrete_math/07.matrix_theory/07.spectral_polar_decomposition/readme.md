# 第七章 矩阵分解（四）：谱分解与极分解

## 1. 几何意义

### 谱分解

对称矩阵 $A$ 可以分解为 $A = Q \Lambda Q^T$，其中 $Q$ 正交，$\Lambda$ 对角。

**几何含义**：对称矩阵的变换效果等价于：旋转到特征向量坐标系 → 沿各轴独立缩放 → 旋转回来。对称矩阵不产生剪切，只产生沿正交方向的拉伸/压缩。

### 极分解

任意可逆矩阵 $A \in \mathbb{R}^{n \times n}$ 可分解为

$$A = QP \quad \text{（右极分解）}$$

其中 $Q$ 正交（旋转/反射），$P = (A^T A)^{1/2}$ 对称正定（纯缩放）。

**几何含义**：极分解将任意线性变换分解为**旋转**和**纯缩放**两部分，类似于复数的极坐标表示 $z = re^{i\theta}$。$P$ 刻画了变换对各方向的拉伸程度，$Q$ 刻画了变换中的旋转成分。

类比：$A$ 像一条橡皮筋被拉伸和扭转，$P$ 是拉伸量，$Q$ 是扭转角度。

## 2. 应用场景

| 领域 | 应用 |
|------|------|
| 弹性力学 | 形变梯度张量的极分解：旋转（刚体转动）+ 拉伸（应变） |
| 计算机视觉 | 本征图像分解将图像分解为反射率和光照 |
| 量子力学 | 密度矩阵的谱分解用于测量概率计算 |
| 协方差分析 | 主成分分析本质上是协方差矩阵的谱分解 |
| 优化理论 | Newton法中Hessian矩阵的特征值判断凸性 |
| 信号处理 | Karhunen-Loeve变换本质上是谱分解 |

## 3. 数学理论

### 3.1 实对称矩阵的谱定理

**定理（谱定理）**：设 $A \in \mathbb{R}^{n \times n}$ 为实对称矩阵，则

$$A = Q \Lambda Q^T = \sum_{i=1}^{n} \lambda_i q_i q_i^T$$

其中 $Q = [q_1, \ldots, q_n]$ 为正交矩阵，$\Lambda = \text{diag}(\lambda_1, \ldots, \lambda_n)$，$\lambda_i$ 为实数。

**等价表述**：$A$ 有 $n$ 个实特征值（计重数），对应 $n$ 个正交特征向量。

### 3.2 谱分解的性质

**谱投影矩阵**：定义 $P_i = q_i q_i^T$，则

- $P_i^2 = P_i$（幂等，投影矩阵）
- $P_i P_j = 0$（$i \neq j$，正交投影）
- $\sum_{i=1}^{n} P_i = I$（完备性）
- $A = \sum_{i=1}^{n} \lambda_i P_i$（谱分解）

**证明完备性**：$Q Q^T = \sum_i q_i q_i^T = I$。

**函数作用**：$f(A) = \sum_{i=1}^{n} f(\lambda_i) P_i = Q \cdot \text{diag}(f(\lambda_1), \ldots, f(\lambda_n)) \cdot Q^T$。

### 3.3 正规矩阵

**定义**：$A \in \mathbb{C}^{n \times n}$ 为**正规矩阵**，若 $A^H A = AA^H$。

正规矩阵包含：实对称矩阵、实反对称矩阵、正交矩阵、Hermitian矩阵、反Hermitian矩阵、酉矩阵、对角矩阵。

**定理（复谱定理）**：$A$ 为正规矩阵的充要条件是 $A$ 可酉对角化：

$$A = U \Lambda U^H$$

其中 $U$ 为酉矩阵，$\Lambda$ 为对角矩阵（元素可以为复数）。

**推论**：
- 实对称矩阵的特征值为实数，可实正交对角化
- 实反对称矩阵的特征值为纯虚数或零
- 正交矩阵的特征值模为 1

### 3.4 正定矩阵的平方根

**定理**：若 $A$ 对称正定，则存在唯一的对称正定矩阵 $B$ 使得 $B^2 = A$，记 $B = A^{1/2}$。

**构造**：设 $A = Q\Lambda Q^T$，取 $A^{1/2} = Q \Lambda^{1/2} Q^T$，其中 $\Lambda^{1/2} = \text{diag}(\sqrt{\lambda_1}, \ldots, \sqrt{\lambda_n})$。

**验证**：

$$(A^{1/2})^2 = Q\Lambda^{1/2} Q^T Q\Lambda^{1/2} Q^T = Q\Lambda Q^T = A \quad \checkmark$$

### 3.5 极分解定理

**定理**：设 $A \in \mathbb{R}^{n \times n}$ 可逆，则存在唯一的正交矩阵 $Q$ 和唯一的对称正定矩阵 $P, S$，使得

$$A = QP \quad \text{（右极分解）}$$
$$A = SQ \quad \text{（左极分解）}$$

其中 $P = (A^T A)^{1/2}$，$S = (AA^T)^{1/2}$，$Q = AP^{-1} = S^{-1}A$。

**证明（右极分解）**：

1. $A^T A$ 为对称正定矩阵（$x^T A^T A x = \|Ax\|^2 > 0$，$x \neq 0$）
2. 定义 $P = (A^T A)^{1/2}$（对称正定）
3. 定义 $Q = AP^{-1}$

验证 $Q$ 为正交矩阵：

$$Q^T Q = P^{-T} A^T A P^{-1} = P^{-1} (A^T A) P^{-1} = P^{-1} P^2 P^{-1} = I$$

**唯一性**：若 $A = Q_1 P_1 = Q_2 P_2$，则 $Q_2^T Q_1 = P_2 P_1^{-1}$。左端正交，右端对称正定，故两者都等于 $I$，得 $Q_1 = Q_2$，$P_1 = P_2$。

### 3.6 极分解与SVD的关系

设 $A = U \Sigma V^T$ 为 SVD，则

- $P = (A^T A)^{1/2} = V \Sigma V^T$
- $S = (AA^T)^{1/2} = U \Sigma U^T$
- $Q = UV^T$

即 $A = (UV^T)(V\Sigma V^T) = US \cdot Q$（对应不同写法）。

### 3.7 应用：矩阵的条件数与数值稳定性

由极分解 $A = QP$，$P$ 的特征值（即 $A^T A$ 的奇异值）反映了变换的各向异性程度：

- 若 $\sigma_{\max}/\sigma_{\min}$ 很大（条件数大），则某些方向被极大地拉伸/压缩，数值求解时误差会被放大
- 条件数 $\kappa(A) = \sigma_1 / \sigma_n$ 度量了 $A$ 的"病态"程度

**例**：Hilbert矩阵 $H_{ij} = 1/(i+j-1)$ 是典型的病态矩阵，$n = 10$ 时条件数约 $10^{13}$。
