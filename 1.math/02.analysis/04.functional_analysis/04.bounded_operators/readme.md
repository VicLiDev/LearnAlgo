# 第四章 有界线性算子

## 几何意义

线性算子是线性代数中矩阵的自然推广。在有限维空间中，每个线性映射都可以用一个矩阵来表示；在无穷维空间中，线性算子可以看作"无穷维矩阵"。

**有界性**的几何意义是：算子将单位球映射为有界集。形象地说，有界线性算子不会将空间"无限拉伸"——存在一个常数 $M$，使得输入向量的长度最多被放大 $M$ 倍。

算子范数 $\|T\|$ 的几何含义是：$T$ 的"最大放大倍数"：

$$
\|T\| = \sup_{\|x\|=1} \|Tx\|
$$

在有限维空间中，每个线性算子都是有界的（连续的）；但在无穷维空间中，存在**无界线性算子**（不连续的线性算子），这是无穷维分析区别于有限维线性代数的关键特征之一。

**线性泛函**是取值为标量的线性算子。Riesz 表示定理表明，Hilbert 空间上的每个有界线性泛函都可以用内积来表示——这与 $\mathbb{R}^n$ 中"每个线性泛函对应一个向量"（通过点积）的结论完全一致。

## 应用场景

1. **微分方程与积分方程**：微分算子和积分算子是泛函分析中最核心的算子类型，许多 PDE 问题可以写成算子方程 $Tu = f$。

2. **数值分析**：矩阵范数用于分析算法的条件数和数值稳定性；迭代法的收敛速度与迭代算子的谱半径密切相关。

3. **量子力学**：物理可观测量用自伴算子表示，量子态的演化由酉算子描述。

4. **信号处理**：线性滤波器是 $L^2$ 空间上的有界线性算子（卷积算子）；Fourier 变换是酉算子。

5. **控制理论**：系统传递函数、状态空间表示都涉及算子理论。

## 数学理论（及推导）

### 4.1 线性算子与有界性

**定义 4.1**（线性算子） 设 $X, Y$ 是同一标量域 $\mathbb{K}$ 上的线性空间。映射 $T: X \to Y$ 称为**线性算子**，若对任意 $x, y \in X$ 和 $\alpha, \beta \in \mathbb{K}$：

$$
T(\alpha x + \beta y) = \alpha Tx + \beta Ty
$$

**定义 4.2**（有界算子） 设 $X, Y$ 是赋范线性空间，线性算子 $T: X \to Y$ 称为**有界的**，若存在常数 $C > 0$ 使得

$$
\|Tx\| \leq C \|x\|, \quad \forall\, x \in X
$$

**注**：此处的"有界"与"将有界集映射为有界集"等价（对线性算子而言）。

**例 4.1**（恒等算子） $Ix = x$，有界且 $\|I\| = 1$。

**例 4.2**（矩阵算子） $A: \mathbb{R}^n \to \mathbb{R}^m$，$Ax = Ax$（矩阵乘法）。$A$ 有界，$\|A\|$ 是矩阵的算子范数。

**例 4.3**（微分算子——无界） $D: C^1[0,1] \to C[0,1]$，$Df = f'$。在 $C^1[0,1]$ 取 $C[0,1]$ 的上确界范数时，$D$ 是无界的。例如取 $f_n(t) = t^n$，$\|f_n\|_\infty = 1$，但 $\|Df_n\|_\infty = n \to \infty$。

**例 4.4**（积分算子——有界） 设 $K \in C([a,b] \times [a,b])$，定义 Volterra 型积分算子：

$$
(Tf)(s) = \int_a^b K(s,t) f(t) \, dt
$$

则 $T: C[a,b] \to C[a,b]$ 有界，且 $\|T\| \leq (b-a) \max_{s,t} |K(s,t)|$。

### 4.2 算子范数

**定义 4.3**（算子范数） 设 $T: X \to Y$ 是有界线性算子，定义 $T$ 的**算子范数**为：

$$
\|T\| = \sup_{x \neq 0} \frac{\|Tx\|}{\|x\|} = \sup_{\|x\|=1} \|Tx\| = \sup_{\|x\| \leq 1} \|Tx\|
$$

**命题 4.1** 算子范数确实满足范数公理：
1. $\|T\| \geq 0$，$\|T\| = 0 \iff T = 0$
2. $\|\alpha T\| = |\alpha| \cdot \|T\|$
3. $\|T + S\| \leq \|T\| + \|S\|$

此外，**复合算子的范数**满足 $\|TS\| \leq \|T\| \cdot \|S\|$。

**证明**（三角不等式）：对 $\|x\| \leq 1$，

$$
\|(T+S)x\| = \|Tx + Sx\| \leq \|Tx\| + \|Sx\| \leq \|T\| + \|S\|
$$

取上确界即得。$\blacksquare$

### 4.3 有界线性算子空间 B(X, Y)

**定理 4.1** 设 $(X, \|\cdot\|_X)$ 和 $(Y, \|\cdot\|_Y)$ 是赋范线性空间，则所有从 $X$ 到 $Y$ 的有界线性算子之集

$$
\mathcal{B}(X,Y) = \{T: X \to Y \mid T \text{ 线性且有界}\}
$$

在算子范数下构成赋范线性空间。若 $Y$ 是 Banach 空间，则 $\mathcal{B}(X,Y)$ 也是 Banach 空间。

**证明**（完备性）：设 $\{T_n\}$ 是 $\mathcal{B}(X,Y)$ 中的 Cauchy 序列。对每个 $x \in X$：

$$
\|T_n x - T_m x\| \leq \|T_n - T_m\| \cdot \|x\| \to 0
$$

故 $\{T_n x\}$ 是 $Y$ 中的 Cauchy 序列，由 $Y$ 的完备性，定义 $Tx = \lim_{n\to\infty} T_n x$。

- **$T$ 的线性**：$T(\alpha x + \beta y) = \lim T_n(\alpha x + \beta y) = \alpha \lim T_n x + \beta \lim T_n y = \alpha Tx + \beta Ty$。
- **$T$ 的有界性**：$\{T_n\}$ 是 Cauchy 序列，故 $\|T_n\| \leq M$（一致有界）。$\|Tx\| = \lim \|T_n x\| \leq M\|x\|$，故 $T$ 有界。
- **$\|T_n - T\| \to 0$**：对任意 $\varepsilon > 0$，存在 $N$ 使得 $n,m \geq N$ 时 $\|T_n - T_m\| < \varepsilon$。对 $\|x\| \leq 1$，令 $m \to \infty$，$\|(T_n - T)x\| \leq \varepsilon$，故 $\|T_n - T\| \leq \varepsilon$。$\blacksquare$

**定义 4.4** 简记 $\mathcal{B}(X) = \mathcal{B}(X,X)$，$\mathcal{B}(X)^*$ 或 $X^* = \mathcal{B}(X, \mathbb{K})$（有界线性泛函空间）。

### 4.4 连续性与有界性的等价

**定理 4.2** 设 $X, Y$ 是赋范线性空间，$T: X \to Y$ 是线性算子。则以下等价：

1. $T$ 连续
2. $T$ 在某点连续（例如 $0$ 点）
3. $T$ 有界

**证明**：

$(3) \Rightarrow (1)$：设 $\|Tx\| \leq M\|x\|$。若 $x_n \to x$，则 $\|Tx_n - Tx\| = \|T(x_n - x)\| \leq M\|x_n - x\| \to 0$。

$(1) \Rightarrow (2)$：显然。

$(2) \Rightarrow (3)$：设 $T$ 在 $0$ 点连续，则对 $\varepsilon = 1$，存在 $\delta > 0$ 使得 $\|x\| < \delta \implies \|Tx\| < 1$。

对任意 $y \neq 0$，令 $x = \frac{\delta}{2\|y\|} y$，则 $\|x\| = \delta/2 < \delta$，故 $\|Tx\| < 1$，即

$$
\left\|T\left(\frac{\delta}{2\|y\|} y\right)\right\| = \frac{\delta}{2\|y\|} \|Ty\| < 1
$$

故 $\|Ty\| < \frac{2}{\delta} \|y\|$。$\blacksquare$

### 4.5 有限维情形：线性算子即矩阵

**命题 4.2** 设 $\dim X = n$，$\dim Y = m$，$\{e_1, \ldots, e_n\}$ 是 $X$ 的基，$\{f_1, \ldots, f_m\}$ 是 $Y$ 的基。则每个线性算子 $T: X \to Y$ 对应唯一的 $m \times n$ 矩阵 $A$ 使得

$$
T\left(\sum_{j=1}^n x_j e_j\right) = \sum_{i=1}^m \left(\sum_{j=1}^n a_{ij} x_j\right) f_i
$$

其中 $a_{ij}$ 由 $Te_j = \sum_{i=1}^m a_{ij} f_i$ 确定。

**命题 4.3**（有限维算子总有界） 有限维空间之间的线性算子是有界的。特别地，对 $\mathbb{K}^{n \times n}$ 上的算子范数 $\|A\| = \sup_{\|x\|=1} \|Ax\|$，有以下等价计算方式：

- $\|A\|_1 = \max_j \sum_i |a_{ij}|$（列和范数）
- $\|A\|_\infty = \max_i \sum_j |a_{ij}|$（行和范数）
- $\|A\|_2 = \sigma_{\max}(A)$（谱范数，即最大奇异值）

### 4.6 线性泛函

**定义 4.5** 取值在标量域 $\mathbb{K}$ 中的有界线性算子 $f: X \to \mathbb{K}$ 称为**有界线性泛函**。全体有界线性泛函构成**对偶空间**（共轭空间）：

$$
X^* = \mathcal{B}(X, \mathbb{K})
$$

**定理 4.3**（有界线性泛函的例子）

**(a)** $\ell^p$ 的对偶空间：设 $1 \leq p < \infty$，$1/p + 1/q = 1$，则 $(\ell^p)^* \cong \ell^q$。具体地，每个 $f \in (\ell^p)^*$ 可以唯一表示为

$$
f(x) = \sum_{n=1}^\infty x_n y_n
$$

其中 $y = (y_n) \in \ell^q$，且 $\|f\| = \|y\|_q$。

**(b)** $L^p[a,b]$ 的对偶空间：$(L^p[a,b])^* \cong L^q[a,b]$（$1 \leq p < \infty$，$1/p + 1/q = 1$），每个 $f \in (L^p)^*$ 表示为

$$
f(g) = \int_a^b g(t) \overline{h(t)} \, dt
$$

其中 $h \in L^q$，$\|f\| = \|h\|_q$。

**(c)** $C[a,b]$ 的对偶空间：$(C[a,b])^*$ 等同于所有正则化有界变差函数构成的 Banach 空间，这是 **Riesz 表示定理**的内容（$C[a,b]$ 的版本，与 Hilbert 空间版本的 Riesz 定理不同）。

### 4.7 算子的逆与等距算子

**定义 4.6**（等距算子） 有界线性算子 $U: X \to Y$ 称为**等距算子**，若 $\|Ux\| = \|x\|$ 对所有 $x \in X$ 成立。

**定义 4.7**（酉算子） Hilbert 空间 $H$ 上的有界线性算子 $U: H \to H$ 称为**酉算子**，若 $U$ 是满射等距算子（即 $U^*U = UU^* = I$）。

**例 4.5** Fourier 变换 $\mathcal{F}: L^2(\mathbb{R}) \to L^2(\mathbb{R})$ 是酉算子（Plancherel 定理）。
