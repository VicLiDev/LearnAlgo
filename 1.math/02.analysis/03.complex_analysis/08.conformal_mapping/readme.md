# 第八章 保形映射

## 一、几何意义

### 1.1 保形映射的基本概念

**定义**：设 $w = f(z)$ 是区域 $D$ 到 $D'$ 的映射。若 $f$ 在 $D$ 内解析且 $f'(z) \neq 0$（$\forall z \in D$），则称 $f$ 为 $D$ 内的**保形映射**（Conformal mapping），也称**共形映射**。

保形映射具有两个核心几何性质：

1. **保角性**（Angle preservation）：在映射前后，通过任一点的任意两条曲线的夹角（大小和转向）保持不变。
2. **伸缩一致性**（Scale uniformity）：在一点附近，各方向的伸缩比相同（均等于 $|f'(z)|$）。

**局部几何**：在 $z_0$ 处，$f(z) \approx f(z_0) + f'(z_0)(z-z_0)$。线性部分 $f'(z_0)(z-z_0)$ 是一个复数乘法，等价于旋转角度 $\arg f'(z_0)$ 和伸缩因子 $|f'(z_0)|$。因此，保形映射在局部近似为旋转加伸缩，保持小图形的形状。

### 1.2 分式线性变换的几何性质

**分式线性变换**（Möbius 变换）：

$$w = T(z) = \frac{az + b}{cz + d}, \quad ad - bc \neq 0$$

几何特性：

1. **圆性**（Circle preservation）：将圆和直线映射为圆或直线（直线视为过无穷远点的圆）。
2. **对称点保持**：若 $z_1$、$z_2$ 关于圆 $C$ 对称，则 $T(z_1)$、$T(z_2)$ 关于 $T(C)$ 对称。
3. **三点确定**：唯一确定一个分式线性变换需要指定三组对应点 $(z_1, w_1)$，$(z_2, w_2)$，$(z_3, w_3)$。
4. **保向性**：当 $ad - bc > 0$ 时保持定向，$ad - bc < 0$ 时反转定向。
5. **保角性**：在全平面（包括 $\infty$）保形。

**分解**：任意 Möbius 变换可以分解为以下基本变换的复合：

$$T(z) = \frac{a}{c} + \frac{bc-ad}{c}\cdot\frac{1}{cz+d}$$

即 **平移**（$z \mapsto z + d/c$）$\to$ **旋转伸缩**（$z \mapsto cz$）$\to$ **反演**（$z \mapsto 1/z$）$\to$ **平移**（$z \mapsto z + a/c$）。

### 1.3 反演变换 $w = 1/z$ 的几何

反演 $w = 1/z$（或写成 $w = \frac{\bar{z}}{|z|^2}$）的几何意义：

- **单位圆上的点**映射到单位圆上（$|z|=1 \Rightarrow |w|=1$）。
- **单位圆内部的点**映射到外部（$|z|<1 \Rightarrow |w|>1$），反之亦然。
- **关于单位圆对称的点对** $z$ 和 $\frac{1}{\bar{z}}$ 映射为关于实轴对称的点对 $w$ 和 $\bar{w}$。
- 上半平面映射到下半平面（或反过来，取决于 $z$ 的位置）。

### 1.4 典型区域间的映射

保形映射的核心应用是将复杂区域上的边值问题转化为简单区域（如上半平面、单位圆盘）上的问题。

常见映射关系：
- 上半平面 $\leftrightarrow$ 单位圆盘
- 带形区域 $\leftrightarrow$ 上半平面
- 角形区域 $\leftrightarrow$ 上半平面
- 多边形区域 $\leftrightarrow$ 上半平面（Schwarz-Christoffel 变换）

## 二、应用场景

### 2.1 二维边值问题的求解

保形映射最重要的应用是将复杂区域上的 Laplace 方程 $\Delta u = 0$ 化为简单区域上的问题。

**物理问题**：
- 静电场中的电势分布（给定导体边界上的电势）。
- 稳态温度分布。
- 流体绕障碍物的流动。

**方法**：寻找保形映射 $w = f(z)$ 将复杂区域 $D$ 映射到简单区域 $D'$（如上半平面）。在 $D'$ 上求解 Dirichlet 问题，再通过逆映射回到 $D$。

### 2.2 空气动力学——Joukowski 变换

Joukowski 变换 $w = z + \frac{c^2}{z}$（$c > 0$）将圆映射为翼型。这是空气动力学理论的基石之一，用于：
- 计算翼型绕流的速度场和压力分布。
- 分析升力（Kutta-Joukowski 定理：升力 $L = \rho V \Gamma$）。

### 2.3 天线设计

保形映射用于计算特定形状天线的辐射模式。通过将复杂截面映射为可分析形状，可以简化电磁场的计算。

### 2.4 图像处理与计算机图形学

保形映射在以下领域有应用：
- 纹理映射：将平面纹理保形地贴到曲面上。
- 医学成像：将大脑皮层表面展开到平面上进行分析。
- 参数化：三维曲面的保形参数化。

### 2.5 制图学

Mercator 投影是一种保形映射，它保持角度但不保持面积。这在航海中非常重要，因为恒定航向角在地图上表现为直线。

## 三、数学理论（及推导）

### 3.1 保形映射的定义与条件

**定理**：设 $f(z)$ 在区域 $D$ 内解析，$z_0 \in D$。

- 若 $f'(z_0) \neq 0$，则 $f$ 在 $z_0$ 处是**保角的**（conformal at $z_0$）。
- 若 $f'(z_0) = 0$，则 $f$ 在 $z_0$ 处**不保角**。此时角度被放大 $m$ 倍，其中 $m$ 是 $f'(z)$ 在 $z_0$ 处零点的阶数。

**保角性的证明**：

设曲线 $C$ 通过 $z_0$，参数化为 $z = z(t)$，$z_0 = z(t_0)$。映射后 $w = f(z(t))$。

切线方向：
$$\frac{dw}{dt}\Big|_{t_0} = f'(z_0)\cdot z'(t_0)$$

设两条曲线 $C_1$、$C_2$ 通过 $z_0$，切向量分别为 $z_1'(t_0)$ 和 $z_2'(t_0)$。映射后切向量为 $f'(z_0)z_1'(t_0)$ 和 $f'(z_0)z_2'(t_0)$。

夹角：

$$\arg[f'(z_0)z_2'(t_0)] - \arg[f'(z_0)z_1'(t_0)] = \arg f'(z_0) + \arg z_2' - \arg f'(z_0) - \arg z_1' = \arg z_2' - \arg z_1'$$

因此夹角（包括转向）保持不变。$\blacksquare$

**当 $f'(z_0) = 0$ 时**：设 $f'(z_0) = 0$ 但 $f''(z_0), \ldots, f^{(m-1)}(z_0) = 0$，$f^{(m)}(z_0) \neq 0$。则：

$$f(z) - f(z_0) \approx \frac{f^{(m)}(z_0)}{m!}(z-z_0)^m$$

角度变为原来的 $m$ 倍。

### 3.2 分式线性变换（Möbius 变换）

**定义**：

$$w = T(z) = \frac{az + b}{cz + d}, \quad ad - bc \neq 0$$

**基本性质**：

1. **可逆性**：$T^{-1}(w) = \frac{dw - b}{-cw + a}$，逆变换也是 Möbius 变换。

2. **群结构**：Möbius 变换在复合运算下构成群（Möbius 群 / PSL(2, $\mathbb{C}$)）。

3. **三点确定唯一变换**：给定不同的 $z_1, z_2, z_3 \in \hat{\mathbb{C}}$ 和 $w_1, w_2, w_3 \in \hat{\mathbb{C}}$，存在唯一 Möbius 变换使 $T(z_k) = w_k$。

**交比**（Cross ratio）：四个不同点 $z_1, z_2, z_3, z_4$ 的交比为：

$$(z_1, z_2, z_3, z_4) = \frac{(z_1-z_3)(z_2-z_4)}{(z_1-z_4)(z_2-z_3)}$$

Möbius 变换保持交比不变。因此，将 $z_1 \mapsto w_1$、$z_2 \mapsto w_2$、$z_3 \mapsto w_3$ 的变换为：

$$\frac{(w - w_1)(w_2 - w_3)}{(w - w_3)(w_2 - w_1)} = \frac{(z - z_1)(z_2 - z_3)}{(z - z_3)(z_2 - z_1)}$$

### 3.3 圆性定理

**定理**：Möbius 变换将广义圆（圆或直线）映射为广义圆。

**证明**：只需验证基本变换具有此性质。
- 平移 $w = z + b$：显然。
- 旋转伸缩 $w = az$：显然。
- 反演 $w = 1/z$：圆的方程为 $A(x^2+y^2) + Bx + Cy + D = 0$，令 $z = x+iy$，$\bar{z} = x-iy$，得：

$$A|z|^2 + \frac{B}{2}(z+\bar{z}) + \frac{C}{2i}(z-\bar{z}) + D = 0$$

令 $w = 1/z$，即 $z = 1/w$，代入：

$$\frac{A}{|w|^2} + \frac{B}{2}\left(\frac{1}{w}+\frac{1}{\bar{w}}\right) + \frac{C}{2i}\left(\frac{1}{w}-\frac{1}{\bar{w}}\right) + D = 0$$

乘以 $|w|^2$：

$$A + \frac{B}{2}(w+\bar{w}) + \frac{C}{2i}(w-\bar{w}) + D|w|^2 = 0$$

这仍是一个广义圆的方程（$A$ 和 $D$ 交换了角色）。$\blacksquare$

### 3.4 对称点与保对称性

**定义**：点 $z_1$ 和 $z_2$ 关于圆 $C$（圆心 $a$，半径 $R$）**对称**，如果：

$$\overline{(z_1 - a)(z_2 - a)} = R^2$$

即 $z_2 = a + \frac{R^2}{\overline{z_1 - a}}$。

- 关于直线的对称：通常的轴对称反射。
- 关于圆 $|z-a|=R$ 的对称：$z_2$ 在射线 $az_1$ 上，$|z_1-a|\cdot|z_2-a| = R^2$。

**定理**：Möbius 变换保持对称点关系。

### 3.5 典型保形映射

**（1）上半平面到单位圆盘**

$$w = e^{i\theta}\frac{z - z_0}{z - \bar{z}_0}, \quad \operatorname{Im}z_0 > 0, \quad \theta \in \mathbb{R}$$

**验证**：当 $z \in \mathbb{R}$（实轴）时，$\bar{z} = z$：

$$|w| = \left|\frac{z - z_0}{z - \bar{z}_0}\right| = \frac{|z - z_0|}{|\overline{z - z_0}|} = 1$$

故实轴映射为单位圆周。当 $z = z_0$（上半平面内的点）时，$w = 0$（单位圆盘内部），故上半平面映射到单位圆盘内部。

**（2）单位圆盘到自身**

$$w = e^{i\theta}\frac{z - z_0}{1 - \bar{z}_0 z}, \quad |z_0| < 1, \quad \theta \in \mathbb{R}$$

**验证**：当 $|z| = 1$ 时，$\bar{z} = 1/z$：

$$|w| = \left|\frac{z - z_0}{1 - \bar{z}_0 z}\right| = \left|\frac{z - z_0}{z(\bar{z} - \bar{z}_0)}\right| = \frac{|z-z_0|}{|z|\cdot|\bar{z}-\bar{z}_0|} = 1$$

当 $z = z_0$ 时，$w = 0$。故单位圆盘映射到自身。

**（3）带形区域到上半平面**

带形 $0 < \operatorname{Im}z < \pi$ 到上半平面的映射为 $w = e^z$。

验证：$z = x + iy$（$0 < y < \pi$），$w = e^x(\cos y + i\sin y)$。$\operatorname{Im}w = e^x\sin y > 0$（因 $0 < y < \pi$）。

更一般地，带形 $0 < \operatorname{Im}z < h$ 到上半平面的映射为 $w = e^{\pi z / h}$。

**（4）角形区域到上半平面**

角形 $0 < \arg z < \alpha$（$\alpha < 2\pi$）到上半平面的映射为：

$$w = z^{\pi/\alpha}$$

验证：$z = re^{i\theta}$（$0 < \theta < \alpha$），$w = r^{\pi/\alpha}e^{i\pi\theta/\alpha}$。$0 < \frac{\pi\theta}{\alpha} < \pi$，故 $\operatorname{Im}w > 0$。

### 3.6 Schwarz-Christoffel 变换

将上半平面映射到多边形内部区域的变换称为 **Schwarz-Christoffel 变换**。

设多边形的顶点为 $w_1, w_2, \ldots, w_n$（按正向排列），对应上半平面上的点 $x_1 < x_2 < \cdots < x_n$。外角分别为 $\alpha_1\pi, \alpha_2\pi, \ldots, \alpha_n\pi$（$-1 < \alpha_k < 1$，$\sum \alpha_k = 2$）。

**Schwarz-Christoffel 公式**：

$$w = f(z) = A + C\int^z \prod_{k=1}^{n}(t - x_k)^{\alpha_k - 1}\,dt$$

其中 $A \in \mathbb{C}$ 为平移，$C \in \mathbb{C}$ 为旋转伸缩。

**推导思路**：$f'(z) = C\prod_{k=1}^{n}(z-x_k)^{\alpha_k - 1}$。在 $x_k$ 附近，$f'(z) \sim (z-x_k)^{\alpha_k-1}$，使得 $\arg f'(z)$ 在 $x_k$ 处跃变 $\alpha_k\pi$，对应多边形在 $w_k$ 处的外角。

### 3.7 Riemann 映射定理

**定理（Riemann 映射定理）**：设 $D$ 是 $\mathbb{C}$ 中非空的真单连通开集，则存在保形映射 $f: D \to \mathbb{D}$（$\mathbb{D}$ 为单位圆盘），且在相差一个 Möbius 自同构的意义下唯一。

**注**：
- "真"指 $D \neq \mathbb{C}$（全平面不能保形映射到单位圆盘，由 Liouville 定理）。
- 唯一性可以加入规范化条件，例如指定 $f(z_0) = 0$，$f'(z_0) > 0$。
- 该定理是保形映射理论的基石，但其证明（通常通过正规族和 Montel 定理）是存在性的，不提供构造方法。

**推论**：任意两个非全平面的单连通区域之间存在保形映射。

**意义**：Riemann 映射定理保证了许多边值问题可以在标准区域（如单位圆盘）上求解，然后通过保形映射回到原区域。这为偏微分方程的求解提供了统一的框架。
