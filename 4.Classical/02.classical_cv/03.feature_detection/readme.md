# 特征检测与描述 (Feature Detection & Description)

## 1. 简介

特征检测是找到图像中的关键点，特征描述是为这些关键点生成唯一的描述向量。好的特征应该具有**不变性**（对旋转、尺度、光照等变化不敏感）。

## 2. 算法对比

| 算法  | 尺度不变 | 旋转不变 | 速度 | 描述子维度 | 专利状态 |
|-------|----------|----------|------|------------|----------|
| SIFT  | ✓        | ✓        | 慢   | 128        | 已过期   |
| SURF  | ✓        | ✓        | 中   | 64/128     | 已过期   |
| ORB   | ✓        | ✓        | 快   | 256        | 免费     |
| BRISK | ✓        | ✓        | 快   | 512        | 免费     |
| AKAZE | ✓        | ✓        | 中   | 486        | 免费     |

## 3. SIFT (Scale-Invariant Feature Transform)

### 3.1 流程

```
1. 尺度空间构建
   - 构建高斯金字塔
   - 计算DoG (Difference of Gaussian)

2. 关键点检测
   - 在DoG金字塔中找极值点
   - 亚像素精确定位
   - 去除低对比度和边缘点

3. 方向分配
   - 计算关键点邻域的梯度直方图
   - 主方向作为关键点方向

4. 描述子生成
   - 16×16邻域分为4×4子区域
   - 每个子区域8方向直方图
   - 4×4×8 = 128维描述子
```

### 3.2 DoG (Difference of Gaussian)

```
D(x, y, σ) = L(x, y, kσ) - L(x, y, σ)

其中 L(x, y, σ) = G(x, y, σ) * I(x, y)
```

## 4. SURF (Speeded-Up Robust Features)

### 4.1 加速策略

- 用**盒式滤波器**近似高斯
- 使用**积分图像**加速卷积
- Haar小波计算方向

### 4.2 速度对比

SURF 比 SIFT 快 3-10 倍

## 5. ORB (Oriented FAST and Rotated BRIEF)

### 5.1 组成

- **oFAST**: 带方向的FAST角点检测
- **rBRIEF**: 旋转不变的BRIEF描述子

### 5.2 特点

- 速度极快（SIFT的100倍）
- 免费开源
- 适合实时应用

## 6. 特征匹配

### 6.1 距离度量

| 距离     | 适用描述子          |
|----------|---------------------|
| 欧氏距离 | SIFT, SURF (浮点)   |
| 汉明距离 | ORB, BRIEF (二进制) |

### 6.2 匹配策略

- **暴力匹配**: 与所有描述子计算距离
- **FLANN**: 近似最近邻搜索，速度快
- **比率测试**: 最优/次优 < 0.75
