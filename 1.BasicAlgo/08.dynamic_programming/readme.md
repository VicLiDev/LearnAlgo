# 动态规划 (Dynamic Programming)

## 1. 简介

动态规划是一种通过把原问题分解为相对简单的子问题来求解复杂问题的方法。适用于有
**重叠子问题**和**最优子结构**性质的问题。

## 2. 核心概念

### 2.1 重叠子问题
子问题被重复计算多次，可以用记忆化或表格存储避免重复计算。

### 2.2 最优子结构
问题的最优解包含子问题的最优解。

### 2.3 状态转移方程
描述问题状态之间关系的递推公式。

## 3. 解题步骤

1. **定义状态**: 确定dp数组的含义
2. **找状态转移方程**: 状态之间的关系
3. **确定初始条件**: 边界情况
4. **确定遍历顺序**: 自顶向下或自底向上
5. **返回结果**: 根据问题要求返回

## 4. 经典问题

### 4.1 斐波那契数列
```
dp[i] = dp[i-1] + dp[i-2]
```

### 4.2 爬楼梯
```
问题: 每次爬1或2阶，n阶楼梯有多少种方法？
dp[i] = dp[i-1] + dp[i-2]
```

### 4.3 01背包问题
```
问题: n个物品，容量W的背包，每个物品只能选一次
dp[i][w] = max(dp[i-1][w], dp[i-1][w-wi] + vi)
```

### 4.4 完全背包问题
```
问题: 每个物品可以选多次
dp[i][w] = max(dp[i-1][w], dp[i][w-wi] + vi)
```

### 4.5 最长公共子序列 (LCS)
```
if s1[i] == s2[j]:
    dp[i][j] = dp[i-1][j-1] + 1
else:
    dp[i][j] = max(dp[i-1][j], dp[i][j-1])
```

### 4.6 最长递增子序列 (LIS)
```
dp[i] = max(dp[j] + 1) for j < i if arr[j] < arr[i]
```

### 4.7 编辑距离
```
if s1[i] == s2[j]:
    dp[i][j] = dp[i-1][j-1]
else:
    dp[i][j] = 1 + min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1])
```

## 5. 优化技巧

### 5.1 空间优化
- 滚动数组: 01背包从一维倒序遍历
- 状态压缩: 位运算表示状态

### 5.2 时间优化
- 二分查找: LIS优化到O(n log n)
- 单调队列: 滑动窗口最大值

## 6. DP vs 其他方法

| 方法       | 特点      | 适用场景     |
|------------|-----------|--------------|
| 暴力递归   | 简单直接  | 小规模问题   |
| 记忆化搜索 | 递归+缓存 | 状态转移复杂 |
| 动态规划   | 迭代填表  | 大规模问题   |
| 贪心       | 局部最优  | 特殊问题     |

## 7. 问题分类

### 7.1 线性DP
- 最大子数组和
- 打家劫舍
- 买卖股票

### 7.2 区间DP
- 矩阵链乘法
- 回文子串
- 石子合并

### 7.3 树形DP
- 树的最大独立集
- 树的直径

### 7.4 状态压缩DP
- 旅行商问题
- 集合覆盖

## 8. 复杂度分析

| 问题         | 时间       | 空间        |
|--------------|------------|-------------|
| 斐波那契数列 | O(n)       | O(1)        |
| 01背包       | O(nW)      | O(W)        |
| LCS          | O(mn)      | O(min(m,n)) |
| LIS          | O(n log n) | O(n)        |
| 编辑距离     | O(mn)      | O(min(m,n)) |
