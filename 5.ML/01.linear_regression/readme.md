# 回归算法 (Regression Algorithms)

## 概述

回归算法是监督学习的一种，用于预测连续值输出。根据输入特征，回归模型学习一个映射函数，将输入映射到连续的输出值。

## 常用回归算法

### 1. 线性回归 (Linear Regression)
- **原理**: 假设目标变量与特征之间存在线性关系
- **公式**: y = wx + b
- **适用场景**: 特征与目标之间呈线性关系
- **优点**: 简单、可解释性强
- **缺点**: 只能拟合线性关系

### 2. 多项式回归 (Polynomial Regression)
- **原理**: 线性回归的扩展，拟合多项式函数
- **公式**: y = w₁x + w₂x² + ... + wₙxⁿ + b
- **适用场景**: 特征与目标之间呈非线性关系
- **优点**: 可以拟合非线性关系
- **缺点**: 容易过拟合

### 3. 岭回归 (Ridge Regression)
- **原理**: 在线性回归基础上加入L2正则化
- **公式**: Loss = MSE + α||w||²
- **适用场景**: 存在多重共线性问题
- **优点**: 防止过拟合，处理共线性
- **缺点**: 不能进行特征选择

### 4. Lasso回归 (Lasso Regression)
- **原理**: 在线性回归基础上加入L1正则化
- **公式**: Loss = MSE + α||w||₁
- **适用场景**: 需要进行特征选择
- **优点**: 可以自动进行特征选择
- **缺点**: 对参数α敏感

### 5. 弹性网络 (Elastic Net)
- **原理**: 结合L1和L2正则化
- **公式**: Loss = MSE + α₁||w||₁ + α₂||w||²
- **适用场景**: 特征数量多且存在相关性
- **优点**: 结合了Lasso和Ridge的优点
- **缺点**: 需要调节两个正则化参数

## 评估指标

### 1. 均方误差 (MSE)
```
MSE = (1/n) * Σ(y_true - y_pred)²
```

### 2. 均方根误差 (RMSE)
```
RMSE = √MSE
```

### 3. 平均绝对误差 (MAE)
```
MAE = (1/n) * Σ|y_true - y_pred|
```

### 4. R²分数 (R-squared)
```
R² = 1 - (SS_res / SS_tot)
```
- R² = 1: 完美预测
- R² = 0: 模型等同于均值预测
- R² < 0: 模型比均值预测还差

## 使用建议

1. **数据预处理**
   - 处理缺失值
   - 特征标准化/归一化
   - 处理异常值

2. **特征工程**
   - 特征选择
   - 特征变换
   - 多项式特征

3. **模型选择**
   - 线性关系：选择线性回归
   - 非线性关系：选择多项式回归或其他非线性模型
   - 存在共线性：选择岭回归
   - 需要特征选择：选择Lasso或弹性网络

4. **参数调优**
   - 使用交叉验证
   - 网格搜索或随机搜索
