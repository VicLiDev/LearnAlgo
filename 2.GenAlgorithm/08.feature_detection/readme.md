# 特征检测 (Feature Detection)

## 1. 简介

特征检测是从图像或点云中提取显著特征点的方法，是计算机视觉、图像处理、3D重建等
领域的基础。通过特征检测，可以从原始数据中提取有意义的信息，用于后续的匹配、识别、
定位等任务。

## 3. 霍夫投票 (Hough Voting)

### 3.1 原理

霍夫投票是一种基于投票机制的参数估计方法，通过将图像空间中的点转换到参数空间，
实现对几何形状的鲁棒检测。

**核心思想**：
- 图像空间中的一个点对应参数空间中的一条曲线
- 参数空间中的一个点对应图像空间中的一条曲线/形状
- 通过投票机制累加，找到参数空间的峰值

### 3.2 直线检测

**参数化**：
```
ρ = x·cos(θ) + y·sin(θ)

其中：
- ρ: 原点到直线的距离
- θ: 垂线与x轴的夹角
```

**流程**：
```
1. 初始化累加器 A(ρ, θ)
2. 对每个边缘点 (x, y):
   对每个 θ:
     计算 ρ = x·cos(θ) + y·sin(θ)
     A(ρ, θ) += 1
3. 检测累加器中的局部极大值
```

### 3.3 圆检测

**参数化**：
```
(x-a)² + (y-b)² = r²
参数空间: (a, b, r)
```

**使用梯度加速**：
```
对每个边缘点及其梯度方向:
  沿梯度方向 ±r 处投票
  复杂度从 O(n·nₐ·n_b·n_r) 降至 O(n·n_r)
```

### 3.4 广义霍夫变换

用于检测任意形状：

```
训练阶段 (构建R-table):
1. 选择参考点 c
2. 对每个边缘点:
   - 计算梯度方向 φ
   - 存储相对向量 r = c - p 到 R-table[φ]

检测阶段:
1. 对每个边缘点:
   - 计算梯度方向 φ
   - 从 R-table[φ] 获取偏移并投票
2. 累加器峰值即为形状位置
```

### 3.5 应用

- 直线/圆/椭圆检测
- 车道线检测
- 工业零件定位
- 3D形状检测
- 物体位姿估计
