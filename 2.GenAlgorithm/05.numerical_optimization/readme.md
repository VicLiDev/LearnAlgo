# 数值优化 (Numerical Optimization)

## 1. 简介

数值优化是寻找函数最优解的迭代方法，广泛应用于机器学习、工程优化、参数估计等领域。

## 2. 优化问题分类

| 类型 | 说明 | 示例 |
|------|------|------|
| 无约束优化 | 无限制条件 | min f(x) |
| 有约束优化 | 有等式/不等式约束 | min f(x) s.t. g(x) ≤ 0 |
| 凸优化 | 目标函数和约束都是凸的 | 线性规划、二次规划 |
| 非凸优化 | 存在多个局部最优 | 神经网络训练 |

## 3. 梯度下降法

### 3.1 基本形式

```
x_{k+1} = x_k - α * ∇f(x_k)

α: 学习率 (步长)
∇f: 梯度
```

### 3.2 变体

| 方法 | 特点 |
|------|------|
| SGD | 随机梯度，适合大数据 |
| Momentum | 加入动量，加速收敛 |
| AdaGrad | 自适应学习率 |
| Adam | 结合Momentum和AdaGrad |

### 3.3 学习率选择

```
太大: 振荡、发散
太小: 收敛慢
合适: 稳定快速收敛

常用策略:
- 固定学习率
- 学习率衰减
- 自适应方法 (Adam)
```

## 4. 牛顿法

### 4.1 原理

使用二阶导数（Hessian矩阵）进行优化。

```
x_{k+1} = x_k - H^{-1}(x_k) * ∇f(x_k)

H: Hessian矩阵 (∂²f/∂x_i∂x_j)
```

### 4.2 优缺点

**优点**:
- 收敛速度快（二次收敛）
- 适合二次函数

**缺点**:
- 需要计算Hessian矩阵
- Hessian可能不可逆
- 计算量大

## 5. 拟牛顿法

### 5.1 思想

用近似矩阵B替代Hessian，避免直接计算。

```
B_{k+1} = B_k + ΔB

BFGS更新:
B_{k+1} = B_k + (y_k * y_k^T)/(y_k^T * s_k) - (B_k * s_k * s_k^T * B_k)/(s_k^T * B_k * s_k)

其中:
s_k = x_{k+1} - x_k
y_k = ∇f(x_{k+1}) - ∇f(x_k)
```

### 5.2 常用方法

| 方法 | 存储量 | 适用规模 |
|------|--------|----------|
| BFGS | O(n²) | 中等 |
| L-BFGS | O(mn) | 大规模 |
| DFP | O(n²) | 较少使用 |

## 6. 共轭梯度法

### 6.1 原理

选择共轭方向进行搜索，对二次函数n步收敛。

```
d_0 = -∇f(x_0)
d_k = -∇f(x_k) + β_k * d_{k-1}

β_k 的计算:
- Fletcher-Reeves: β_k = ||∇f_k||² / ||∇f_{k-1}||²
- Polak-Ribiere: β_k = ∇f_k^T(∇f_k - ∇f_{k-1}) / ||∇f_{k-1}||²
```

### 6.2 特点

- 存储量小（只需几个向量）
- 适合大规模问题
- 无需矩阵求逆

## 7. 约束优化

### 7.1 拉格朗日乘数法

```
min f(x)
s.t. g(x) = 0

L(x, λ) = f(x) + λ * g(x)

最优条件: ∇_x L = 0, ∇_λ L = 0
```

### 7.2 KKT条件

```
min f(x)
s.t. g(x) ≤ 0, h(x) = 0

KKT条件:
1. ∇f(x*) + Σ λ_i ∇g_i(x*) + Σ μ_j ∇h_j(x*) = 0
2. λ_i * g_i(x*) = 0 (互补松弛)
3. λ_i ≥ 0
4. g_i(x*) ≤ 0
5. h_j(x*) = 0
```

## 8. 最小二乘

### 8.1 线性最小二乘

```
min ||Ax - b||²

解析解: x = (A^T A)^{-1} A^T b
```

### 8.2 非线性最小二乘

```
min Σ r_i(x)²

高斯-牛顿法:
x_{k+1} = x_k - (J^T J)^{-1} J^T r

其中 J 是残差雅可比矩阵
```
