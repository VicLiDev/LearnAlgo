# 插值算法 (Interpolation)

## 1. 简介

插值是根据已知数据点估计未知点值的方法，广泛应用于图像缩放、数据平滑、曲线拟合等场景。

## 2. 一维插值

| 方法 | 连续性 | 特点 |
|------|--------|------|
| 最近邻 | C⁻¹ | 简单，有锯齿 |
| 线性插值 | C⁰ | 连续，折线 |
| 三次样条 | C² | 平滑，常用 |
| 拉格朗日 | C∞ | 高次多项式 |

## 3. 线性插值

### 3.1 公式

```
已知点 (x0, y0), (x1, y1)

y = y0 + (y1 - y0) * (x - x0) / (x1 - x0)
```

### 3.2 特点

- 计算简单
- 连续但导数不连续
- 适合变化平缓的数据

## 4. 双线性插值 (2D)

### 4.1 原理

对2D网格进行两次线性插值。

```
已知4个点:
Q11(x1,y1)  Q12(x1,y2)
Q21(x2,y1)  Q22(x2,y2)

步骤:
1. 沿x方向插值: R1 = lerp(Q11, Q21), R2 = lerp(Q12, Q22)
2. 沿y方向插值: P = lerp(R1, R2)
```

### 4.2 公式

```
f(x,y) = (1-s)(1-t)*f(0,0) + s(1-t)*f(1,0) + (1-s)t*f(0,1) + st*f(1,1)

其中 s = (x-x1)/(x2-x1), t = (y-y1)/(y2-y1)
```

## 5. 双三次插值

### 5.1 原理

使用4×4邻域的16个点进行插值。

### 5.2 双三次核函数

```
W(x) = (a+2)|x|³ - (a+3)|x|² + 1          (|x| ≤ 1)
     = a|x|³ - 5a|x|² + 8a|x| - 4a        (1 < |x| < 2)
     = 0                                   (|x| ≥ 2)

a通常取-0.5 (Keys' cubic)
```

### 5.3 特点

- 比双线性更平滑
- 计算量较大
- 常用图像缩放

## 6. 三次样条插值

### 6.1 原理

用分段三次多项式拟合，保证二阶导数连续。

### 6.2 自然样条条件

```
S(x) = a_i + b_i(x-x_i) + c_i(x-x_i)² + d_i(x-x_i)³

边界条件:
S''(x_0) = S''(x_n) = 0  (自然样条)
```

### 6.3 求解

```
建立三对角方程组求解系数
复杂度: O(n)
```

## 7. 图像插值对比

| 方法 | 质量 | 速度 | 边缘保持 |
|------|------|------|----------|
| 最近邻 | 差 | 最快 | 锯齿 |
| 双线性 | 中 | 快 | 模糊 |
| 双三次 | 好 | 中 | 较好 |
| Lanczos | 最好 | 慢 | 好 |
