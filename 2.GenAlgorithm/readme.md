## 01. 傅里叶变换 (Fourier Transform)

**难度**: ★★★☆☆

**内容**:
- DFT/FFT原理
- 频域分析
- 信号滤波
- 图像处理应用

**核心公式**:
```
F(k) = Σ f(n) * e^(-2πikn/N)
```

**适用场景**: 信号处理、图像压缩、音频分析

## 02. 旋转矩阵 (Rotation Matrix)

**难度**: ★★☆☆☆

**内容**:
- 2D/3D旋转
- 欧拉角
- 四元数
- 刚体变换
- 坐标系转换

**2D旋转矩阵**:
```
R(θ) = [cos(θ)  -sin(θ)]
       [sin(θ)   cos(θ)]
```

**适用场景**: 计算机图形学、机器人运动学、图像配准

## 03. 上/下采样 (Up/Down Sampling)

**难度**: ★★☆☆☆

**内容**:
- 插值方法（最近邻、双线性、双三次）
- 降采样策略
- 抗混叠滤波
- 多尺度处理

**适用场景**: 图像缩放、点云处理、信号重采样

---

## 04. 几何算法 (Geometric Algorithms)

**难度**: ★★★☆☆

| 算法    | 用途         | 复杂度          |
|---------|--------------|-----------------|
| RANSAC  | 鲁棒模型估计 | O(k·n)          |
| 凸包    | 计算凸多边形 | O(n log n)      |
| KD-Tree | 加速空间搜索 | O(log n)        |
| 最近邻  | 找最近点     | O(n) / O(log n) |

**适用场景**: 点云配准、3D重建、特征匹配

---

## 05. 数值优化 (Numerical Optimization)

**难度**: ★★★☆☆

| 方法     | 收敛速度 | 存储量 |
|----------|----------|--------|
| 梯度下降 | 线性     | O(n)   |
| 牛顿法   | 二次     | O(n²)  |
| L-BFGS   | 超线性   | O(mn)  |
| 共轭梯度 | 超线性   | O(n)   |

**核心公式**:
```
梯度下降: x_{k+1} = x_k - α·∇f(x_k)
牛顿法:   x_{k+1} = x_k - H^{-1}·∇f(x_k)
```

**适用场景**: 机器学习、参数估计、函数拟合

---

## 06. 插值算法 (Interpolation)

**难度**: ★★☆☆☆

| 方法     | 连续性 | 应用      |
|----------|--------|-----------|
| 最近邻   | C⁻¹    | 简单场景  |
| 双线性   | C⁰    | 图像缩放   |
| 三次样条 | C²    | 曲线拟合   |
| 双三次   | C¹    | 高质量缩放 |

**适用场景**: 图像处理、数据平滑、动画插值

---

## 07. 矩阵运算 (Matrix Operations)

**难度**: ★★★☆☆

| 分解     | 形式       | 应用       |
|----------|------------|------------|
| SVD      | A = UΣVᵀ   | 降维、伪逆 |
| 特征分解 | A = PDP⁻¹  | PCA        |
| LU       | A = LU     | 线性方程组 |
| QR       | A = QR     | 最小二乘   |
| Cholesky | A = LLᵀ    | 正定矩阵   |

**适用场景**: 机器学习、信号处理、科学计算
