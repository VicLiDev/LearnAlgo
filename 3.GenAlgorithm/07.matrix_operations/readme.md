# 矩阵运算 (Matrix Operations)

## 1. 简介

矩阵运算是线性代数的基础，广泛应用于机器学习、计算机图形学、信号处理等领域。

## 2. 矩阵分解

| 分解 | 形式 | 应用 |
|------|------|------|
| LU分解 | A = LU | 线性方程组 |
| QR分解 | A = QR | 最小二乘 |
| SVD | A = UΣVᵀ | 降维、伪逆 |
| 特征分解 | A = PDP⁻¹ | 主成分分析 |
| Cholesky | A = LLᵀ | 正定矩阵 |

## 3. SVD (奇异值分解)

### 3.1 定义

```
A = U Σ V^T

A: m×n 矩阵
U: m×m 正交矩阵 (左奇异向量)
Σ: m×n 对角矩阵 (奇异值)
V: n×n 正交矩阵 (右奇异向量)
```

### 3.2 性质

```
1. 奇异值 σ₁ ≥ σ₂ ≥ ... ≥ σ_r > 0
2. 秩 r = 非零奇异值个数
3. 条件数 = σ_max / σ_min
```

### 3.3 应用

```
降维 (低秩近似):
A_k = U_k Σ_k V_k^T

保留前k个最大奇异值
误差: ||A - A_k||_F = σ_{k+1}
```

## 4. 特征值分解

### 4.1 定义

```
A = P D P^{-1}

D: 特征值对角矩阵
P: 特征向量矩阵
```

### 4.2 对称矩阵

```
对称矩阵的特征值都是实数
可以正交对角化: A = Q D Q^T

Q是正交矩阵
```

### 4.3 应用

```
PCA:
1. 计算协方差矩阵 C = X^T X / n
2. 特征分解 C = V D V^T
3. 取前k个最大特征值对应的特征向量
```

## 5. LU分解

### 5.1 定义

```
A = L U

L: 下三角矩阵 (对角线为1)
U: 上三角矩阵
```

### 5.2 求解线性方程组

```
Ax = b
=> LUx = b
=> Ly = b (前代)
=> Ux = y (回代)

复杂度: O(n³) 分解 + O(n²) 求解
```

## 6. QR分解

### 6.1 定义

```
A = Q R

Q: 正交矩阵 (Q^T Q = I)
R: 上三角矩阵
```

### 6.2 计算方法

| 方法 | 复杂度 | 稳定性 |
|------|--------|--------|
| Gram-Schmidt | 2mn² | 差 |
| Householder | 2mn² - 2/3 n³ | 好 |
| Givens | 3mn² | 好 |

## 7. Cholesky分解

### 7.1 定义

```
对称正定矩阵 A = L L^T

L: 下三角矩阵
```

### 7.2 计算复杂度

```
O(n³/3) (比LU分解快一倍)

只适用于对称正定矩阵
```

## 8. 矩阵求逆

### 8.1 方法

```
1. 伴随矩阵法: A^{-1} = adj(A) / det(A)
2. 高斯消元: [A | I] → [I | A^{-1}]
3. LU分解: A^{-1} = U^{-1} L^{-1}
```

### 8.2 伪逆 (Moore-Penrose)

```
A⁺ = V Σ⁺ U^T

Σ⁺: 奇异值取倒数后转置

应用:
- 非方阵求逆
- 最小二乘解: x = A⁺ b
```

## 9. 范数

### 9.1 向量范数

```
||x||_1 = Σ |x_i|
||x||_2 = √(Σ x_i²)
||x||_∞ = max |x_i|
```

### 9.2 矩阵范数

```
||A||_F = √(Σ |a_ij|²)  (Frobenius)
||A||_2 = σ_max         (谱范数)
||A||_1 = max_j Σ |a_ij| (列和)
```
