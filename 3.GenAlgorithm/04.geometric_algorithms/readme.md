## 3. RANSAC (随机采样一致性)

### 3.1 原理

通过随机采样估计模型参数，迭代找出最佳模型。

### 3.2 流程

```
输入: 数据点集, 模型类型, 迭代次数k, 阈值t, 最小内点数d

1. 从数据中随机选取最小点集
2. 用选取的点拟合模型
3. 计算所有点到模型的距离
4. 统计内点数(距离<阈值)
5. 如果内点数>d，更新最佳模型
6. 重复k次，返回最佳模型
```

### 3.3 迭代次数计算

```
k = log(1-p) / log(1-w^n)

其中:
- p: 期望成功率
- w: 内点比例
- n: 最小采样点数
```

### 3.4 应用

- 直线/平面拟合
- 单应性矩阵估计
- 基础矩阵估计
- 点云配准

## 4. 凸包算法

### 4.1 定义

包含所有点的最小凸多边形/多面体。

### 4.2 2D凸包 - Graham扫描

```
1. 找最低点作为起点
2. 按极角排序所有点
3. 依次处理每个点:
   - 如果形成右转，弹出栈顶
   - 否则压入栈
4. 栈中即为凸包顶点
```

### 4.3 复杂度

| 算法 | 2D | 3D |
|------|-----|-----|
| Graham扫描 | O(n log n) | - |
| QuickHull | O(n log n) | O(n log n) |
| 增量法 | O(n²) | O(n²) |

## 5. 最近邻搜索

### 5.1 暴力搜索

```
对每个查询点，遍历所有点找最小距离
复杂度: O(n²)
```

### 5.2 KD-Tree

```
构建:
1. 选择方差最大的维度
2. 按该维度中位数分割
3. 递归构建子树

查询:
1. 从根向下找叶节点
2. 回溯检查是否有更近的点
3. 检查另一子树是否可能更近

平均复杂度: O(log n)
最坏复杂度: O(n)
```

## 6. 点云配准基础

### 6.1 问题定义

```
给定源点云P和目标点云Q
求变换T使得 T(P) ≈ Q
```

### 6.2 SVD求解

```
1. 计算质心: p̄ = mean(P), q̄ = mean(Q)
2. 去中心化: P' = P - p̄, Q' = Q - q̄
3. 计算H矩阵: H = P'^T × Q'
4. SVD分解: H = U Σ V^T
5. 旋转矩阵: R = V × U^T
6. 平移向量: t = q̄ - R × p̄
```
