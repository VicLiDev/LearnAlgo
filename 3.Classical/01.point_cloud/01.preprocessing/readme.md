# 点云预处理 (Point Cloud Preprocessing)

## 1. 简介

点云预处理是3D视觉流程中的重要环节，用于降低噪声、减少数据量、提高后续处理效率。

## 2. 常用预处理方法

### 2.1 降采样 (Downsampling)

| 方法 | 说明 | 适用场景 |
|------|------|----------|
| 体素滤波 (Voxel Grid) | 将空间划分为体素，用体素中心代表所有点 | 大规模点云压缩 |
| 随机采样 | 随机选取一定比例的点 | 快速降采样 |
| 均匀采样 | 按空间均匀分布采样 | 保持空间分布 |

### 2.2 滤波 (Filtering)

| 方法 | 说明 | 适用场景 |
|------|------|----------|
| 统计离群点移除 (SOR) | 基于邻域距离统计移除离群点 | 去除噪声点 |
| 半径离群点移除 | 移除邻域点数少于阈值的点 | 去除孤立点 |
| 双边滤波 | 保边平滑 | 平滑噪声同时保持边缘 |
| 移动最小二乘 (MLS) | 平滑并上采样 | 表面重建预处理 |

### 2.3 裁剪 (Cropping)

| 方法 | 说明 |
|------|------|
| Box裁剪 | 保留指定立方体内的点 |
| 球形裁剪 | 保留指定球体内的点 |
| 平面裁剪 | 保留平面某一侧的点 |

## 3. 算法详解

### 3.1 体素滤波 (Voxel Grid Filter)

```
1. 将空间划分为大小为 v 的体素网格
2. 对每个非空体素:
   - 计算体素内所有点的质心
   - 用质心替代所有点
3. 输出所有质心
```

**参数**: 体素大小 v
**复杂度**: O(n)

### 3.2 统计离群点移除 (Statistical Outlier Removal)

```
1. 对每个点 p_i:
   a. 找 k 个最近邻
   b. 计算到 k 个邻居的平均距离 d_i
2. 计算所有平均距离的均值 μ 和标准差 σ
3. 移除满足 d_i > μ + α*σ 的点
```

**参数**: 邻居数 k, 标准差倍数 α
**复杂度**: O(n log n) (使用KDTree)

### 3.3 半径离群点移除 (Radius Outlier Removal)

```
1. 对每个点 p_i:
   a. 统计半径 r 内的邻居数 n_i
   b. 如果 n_i < min_neighbors, 标记为离群点
2. 移除所有离群点
```

**参数**: 半径 r, 最小邻居数
**复杂度**: O(n log n)

## 4. 法向量估计

### 4.1 PCA方法

```
1. 对点 p 找 k 个最近邻
2. 计算邻域的协方差矩阵
3. 最小特征值对应的特征向量即为法向量
4. 确定法向量方向（通常朝向视点）
```

**参数**: 邻居数 k
**复杂度**: O(n log n)

## 6. 处理流程建议

```
原始点云
    ↓
1. 裁剪 (去除无关区域)
    ↓
2. 离群点移除 (去噪)
    ↓
3. 体素滤波 (降采样)
    ↓
4. 法向量估计
    ↓
处理后点云 → 用于ICP/NDT/PPF等
```

## 7. 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 点云过于稀疏 | 传感器距离太远 | 增大体素尺寸 |
| 边缘模糊 | 体素滤波过度 | 减小体素尺寸或使用MLS |
| 有效点被误删 | SOR参数过严 | 放宽阈值 |
| 法向量方向不一致 | 视点选择问题 | 统一视点方向 |
