# 特征描述子 (Feature Descriptors)

## 1. 简介

特征描述子用于描述图像局部区域的外观特征，是物体检测和识别的基础。

## 2. 算法对比

| 描述子 | 类型 | 维度 | 应用场景 |
|--------|------|------|----------|
| HOG | 梯度直方图 | ~3780 | 行人检测 |
| LBP | 纹理特征 | 256 | 人脸识别 |
| SIFT | 梯度直方图 | 128 | 通用特征 |
| ORB | 二进制 | 256 | 实时应用 |

## 3. HOG (Histogram of Oriented Gradients)

### 3.1 原理

统计图像局部区域的梯度方向分布。

### 3.2 流程

```
1. 预处理
   - 灰度化
   - Gamma校正

2. 计算梯度
   - Sobel计算Gx, Gy
   - 计算梯度幅值和方向

3. 划分Cell
   - 图像划分为小cell (如8×8)
   - 每个cell计算9方向直方图

4. Block归一化
   - 2×2 cells组成block
   - L2归一化

5. 拼接描述子
   - 所有block直方图拼接
```

### 3.3 参数

| 参数 | 说明 | 典型值 |
|------|------|--------|
| cell_size | Cell大小 | 8×8 |
| block_size | Block大小 | 2×2 cells |
| nbins | 方向bin数 | 9 |
| block_stride | Block步长 | 8像素 |

### 3.4 维度计算

```
对于64×128图像:
- cell数: (64/8) × (128/8) = 8 × 16 = 128
- block数: (8-1) × (16-1) = 105
- 描述子维度: 105 × 36 = 3780
```

## 4. LBP (Local Binary Pattern)

### 4.1 原理

比较中心像素与邻域像素的灰度值，生成二进制编码。

### 4.2 基本LBP

```
对于8邻域:
- 如果 neighbor >= center: 1
- 否则: 0

LBP = Σ 2^i * s(neighbor_i - center)
```

### 4.3 改进版本

| 变体 | 说明 |
|------|------|
| Uniform LBP | 0-1跳变≤2的模式 |
| Circular LBP | 圆形邻域，任意半径 |
| Multi-scale LBP | 多尺度组合 |

### 4.4 特点

- 计算极快
- 灰度不变性
- 适合纹理分析
