# 基础滤波与边缘检测 (Basic Filtering & Edge Detection)

## 1. 简介

图像滤波是计算机视觉的基础操作，用于去噪、平滑、锐化等。边缘检测则用于提取图像中物体的轮廓。

## 2. 滤波算法

### 2.1 线性滤波

| 滤波器 | 说明 | 应用 |
|--------|------|------|
| 均值滤波 | 邻域像素平均值 | 简单平滑 |
| 高斯滤波 | 高斯加权平均 | 去噪保边 |
| 双边滤波 | 空间+亮度加权 | 保边平滑 |

### 2.2 非线性滤波

| 滤波器 | 说明 | 应用 |
|--------|------|------|
| 中值滤波 | 邻域像素中值 | 椒盐噪声 |
| 形态学滤波 | 膨胀/腐蚀 | 二值图像处理 |

## 3. 边缘检测

### 3.1 一阶微分算子

| 算子 | 说明 |
|------|------|
| Sobel | 结合高斯平滑的微分 |
| Prewitt | 简单微分 |
| Roberts | 对角微分 |

### 3.2 Canny边缘检测

**流程**:
```
1. 高斯滤波去噪
2. Sobel计算梯度
3. 非极大值抑制
4. 双阈值检测
5. 边缘连接
```

**参数**:
- 低阈值: 弱边缘
- 高阈值: 强边缘
- 介于两者之间: 与强边缘连接则保留

## 4. 数学原理

### 高斯滤波器

```
G(x, y) = (1 / 2πσ²) * exp(-(x² + y²) / 2σ²)
```

### Sobel算子

```
Gx = [-1  0  1]    Gy = [-1 -2 -1]
     [-2  0  2]         [ 0  0  0]
     [-1  0  1]         [ 1  2  1]
```

### 梯度计算

```
G = √(Gx² + Gy²)
θ = arctan(Gy / Gx)
```
