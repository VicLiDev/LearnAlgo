# 角点检测 (Corner Detection)

## 1. 简介

角点是图像中两个边缘的交点，是局部窗口内各方向梯度都很大的点。角点检测是特征提取的基础。

## 2. 算法对比

| 算法 | 速度 | 不变性 | 特点 |
|------|------|--------|------|
| Harris | 中等 | 旋转 | 经典角点检测 |
| Shi-Tomasi | 中等 | 旋转 | Harris改进版 |
| FAST | 快 | 旋转 | 实时检测 |

## 3. Harris角点检测

### 3.1 原理

对于图像窗口 W 内的像素，移动 (u, v) 后的灰度变化：

```
E(u, v) = Σ_{(x,y)∈W} w(x,y) [I(x+u, y+v) - I(x,y)]²
```

使用泰勒展开近似：

```
E(u, v) ≈ [u v] M [u]
                [v]

M = Σ w(x,y) [Ix²   IxIy]
              [IxIy  Iy² ]
```

### 3.2 角点响应

```
R = det(M) - k * trace(M)²
  = λ₁ * λ₂ - k * (λ₁ + λ₂)²
```

- R > 0: 角点
- R < 0: 边缘
- |R| 很小: 平坦区域

### 3.3 参数

| 参数 | 说明 | 典型值 |
|------|------|--------|
| k | 敏感度常数 | 0.04 - 0.06 |
| blockSize | 邻域大小 | 2 - 5 |
| ksize | Sobel核大小 | 3 |

## 4. Shi-Tomasi角点

改进 Harris 的响应函数：

```
R = min(λ₁, λ₂)
```

比 Harris 更稳定，OpenCV 的 `goodFeaturesToTrack` 使用此方法。

## 5. FAST (Features from Accelerated Segment Test)

### 5.1 原理

对于像素 p，检测其周围圆周上的16个像素：
- 如果有 n 个连续像素都亮于或暗于 p，则为角点
- 通常 n = 9 (FAST-9) 或 n = 12 (FAST-12)

### 5.2 加速策略

1. 先检测位置1, 5, 9, 13
2. 如果至少3个满足条件，才检测其余点
3. 使用机器学习优化检测顺序

### 5.3 特点

- 速度极快
- 适合实时应用
- 但不具有尺度不变性
