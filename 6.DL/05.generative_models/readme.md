# 生成模型 (Generative Models)

## 1. 简介

生成模型学习数据分布，并能生成新的样本。主要包括VAE、GAN、Diffusion等。

## 2. 自编码器 (Autoencoder)

### 2.1 基本结构
```
编码器: z = f(x)
解码器: x̂ = g(z)
损失: L = ||x - x̂||²
```

### 2.2 应用
- 降维
- 去噪
- 异常检测

## 3. 变分自编码器 (VAE)

### 3.1 原理
学习隐变量的分布，而非固定编码。

### 3.2 损失函数
```
L = Reconstruction Loss + KL Divergence
L = E[log p(x|z)] - KL(q(z|x) || p(z))
```

### 3.3 重参数化技巧
```
z = μ + σ ⊙ ε,  ε ~ N(0, I)
```

## 4. 生成对抗网络 (GAN)

### 4.1 结构
```
生成器 G: 噪声 z -> 生成图像 G(z)
判别器 D: 图像 x -> 真/假概率 D(x)
```

### 4.2 目标函数
```
min_G max_D V(D,G) = E[log D(x)] + E[log(1-D(G(z)))]
```

### 4.3 常见问题
- 模式崩溃
- 训练不稳定
- 需要仔细调参

### 4.4 改进版本
- **DCGAN**: 使用卷积网络
- **WGAN**: Wasserstein距离
- **StyleGAN**: 风格控制

## 5. Diffusion模型

### 5.1 前向过程
```
x_t = √(α_t) * x_{t-1} + √(1-α_t) * ε
```
逐步添加噪声。

### 5.2 反向过程
```
x_{t-1} = (1/√α_t)(x_t - (1-α_t)/√(1-ᾱ_t) * ε_θ(x_t)) + σ_t * z
```
学习去噪。

### 5.3 优势
- 训练稳定
- 生成质量高
- 理论基础扎实

## 6. 模型对比

| 模型      | 生成质量 | 训练稳定性 | 多样性 | 速度 |
|-----------|----------|------------|--------|------|
| VAE       | 中       | 高         | 高     | 快   |
| GAN       | 高       | 低         | 低     | 快   |
| Diffusion | 最高     | 最高       | 高     | 慢   |

## 7. 评估指标

### 7.1 Inception Score (IS)
```
IS = exp(E_x[KL(p(y|x) || p(y))])
```

### 7.2 FID (Fréchet Inception Distance)
```
FID = ||μ_r - μ_g||² + Tr(Σ_r + Σ_g - 2√(Σ_r·Σ_g))
```
越低越好。

## 8. 应用

- 图像生成
- 风格迁移
- 数据增强
- 图像修复
- 文本到图像
