# 图神经网络 (Graph Neural Networks)

## 1. 简介

图神经网络处理图结构数据，如社交网络、分子结构、知识图谱等。

## 2. 图基础

### 2.1 图表示
- G = (V, E)
- 节点特征矩阵 X ∈ R^{n×d}
- 邻接矩阵 A ∈ R^{n×n}

### 2.2 度矩阵
```
D_ii = Σ_j A_ij
```

## 3. GCN (图卷积网络)

### 3.1 谱方法
```
H = D^(-1/2) A D^(-1/2) X W
```

### 3.2 空间方法
```
h_v^(l+1) = σ(Σ_{u∈N(v)} W h_u^(l) + b)
```

## 4. GAT (图注意力网络)

### 4.1 注意力机制
```
e_ij = a(Wh_i, Wh_j)
α_ij = softmax_j(e_ij)
h'_i = σ(Σ_j α_ij Wh_j)
```

### 4.2 多头注意力
```
h'_i = ||_{k=1}^K σ(Σ_j α_ij^k W^k h_j)
```

## 5. 消息传递框架

```
m_v^(l+1) = Σ_{u∈N(v)} M(h_u^(l), h_v^(l), e_uv)
h_v^(l+1) = U(h_v^(l), m_v^(l+1))
```

### 5.1 常用消息函数
- GCN: M(h_u, h_v) = h_u
- GAT: M(h_u, h_v) = α_uv * h_u
- GraphSAGE: M(h_u, h_v) = [h_u; h_v]

## 6. GraphSAGE

### 6.1 采样策略
```
h_v^(l+1) = σ(W * [h_v^(l); AGG({h_u^(l), u∈N(v)})])
```

### 6.2 聚合函数
- Mean: 均值
- Pool: 最大池化
- LSTM: 序列聚合

## 7. 任务类型

### 7.1 节点分类
预测节点标签。

### 7.2 链接预测
预测边的存在。

### 7.3 图分类
预测整个图的属性。

## 8. 应用

- 社交网络分析
- 推荐系统
- 分子性质预测
- 交通预测
- 知识图谱
