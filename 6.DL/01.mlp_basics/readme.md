# 多层感知机基础 (MLP Basics)

## 1. 简介

多层感知机(Multi-Layer Perceptron)是最基础的深度神经网络，由输入层、隐藏层和输出层组成。

## 2. 核心组件

### 2.1 神经元模型
```
y = f(Σ w_i * x_i + b)
```
- w: 权重
- b: 偏置
- f: 激活函数

### 2.2 前向传播
```
h1 = f(W1 * x + b1)
h2 = f(W2 * h1 + b2)
y = W3 * h2 + b3
```

### 2.3 反向传播
利用链式法则计算梯度：
```
∂L/∂W = ∂L/∂y * ∂y/∂W
```

## 3. 激活函数

| 函数       | 公式                  | 特点                 |
|------------|-----------------------|----------------------|
| Sigmoid    | 1/(1+e^-x)            | 输出(0,1)，梯度消失  |
| Tanh       | (e^x-e^-x)/(e^x+e^-x) | 输出(-1,1)           |
| ReLU       | max(0,x)              | 计算快，缓解梯度消失 |
| Leaky ReLU | max(0.01x,x)          | 解决ReLU死神经元     |
| Softmax    | e^x_i/Σe^x_j          | 多分类输出           |

## 4. 损失函数

### 4.1 回归
- MSE: `L = (1/n)Σ(y-ŷ)²`
- MAE: `L = (1/n)Σ|y-ŷ|`

### 4.2 分类
- 交叉熵: `L = -Σ y*log(ŷ)`
- 二元交叉熵: `L = -[y*log(ŷ) + (1-y)*log(1-ŷ)]`

## 5. 优化算法

### 5.1 SGD
```
w = w - lr * ∂L/∂w
```

### 5.2 Momentum
```
v = β*v + ∂L/∂w
w = w - lr * v
```

### 5.3 Adam
```
m = β1*m + (1-β1)*∂L/∂w
v = β2*v + (1-β2)*(∂L/∂w)²
w = w - lr * m/(√v + ε)
```

## 6. 正则化

### 6.1 L2正则化
```
L_reg = L + λ*||w||²
```

### 6.2 Dropout
训练时随机丢弃神经元，防止过拟合。

## 7. 权重初始化

| 方法   | 公式                                     | 适用场景     |
|--------|------------------------------------------|--------------|
| Xavier | w ~ U(-√6/(n_in+n_out), √6/(n_in+n_out)) | Tanh/Sigmoid |
| He     | w ~ N(0, √2/n_in)                        | ReLU         |

## 8. 网络结构设计

- **输入层**: 特征维度
- **隐藏层**: 通常2-4层，每层64-512神经元
- **输出层**:
  - 回归: 1个神经元，无激活
  - 二分类: 1个神经元，Sigmoid
  - 多分类: n_classes个神经元，Softmax
