# 优化算法 (Optimization)

## 1. 简介

优化算法是训练神经网络的核心，决定了模型能否高效收敛到最优解。

## 2. 梯度下降变体

### 2.1 批量梯度下降 (BGD)
```
θ = θ - η * ∇_θ J(θ)
```
- 使用全部数据计算梯度
- 收敛稳定但慢

### 2.2 随机梯度下降 (SGD)
```
θ = θ - η * ∇_θ J(θ; x_i, y_i)
```
- 每次用一个样本
- 快速但方差大

### 2.3 小批量梯度下降 (Mini-batch GD)
```
θ = θ - η * ∇_θ J(θ; x_{i:i+b}, y_{i:i+b})
```
- 平衡效率和稳定性
- 实践中最常用

## 3. 动量法

### 3.1 SGD with Momentum
```
v_t = γ * v_{t-1} + η * ∇_θ J(θ)
θ = θ - v_t
```
- γ 通常为0.9
- 加速收敛，减少震荡

### 3.2 Nesterov动量
```
v_t = γ * v_{t-1} + η * ∇_θ J(θ - γ * v_{t-1})
θ = θ - v_t
```
- 先"展望"再计算梯度

## 4. 自适应学习率

### 4.1 AdaGrad
```
G_t = G_{t-1} + (∇_θ J(θ))²
θ = θ - η / √(G_t + ε) * ∇_θ J(θ)
```
- 自动调整每个参数的学习率
- 问题：G_t单调递增

### 4.2 RMSprop
```
E[g²]_t = β * E[g²]_{t-1} + (1-β) * (∇_θ J(θ))²
θ = θ - η / √(E[g²]_t + ε) * ∇_θ J(θ)
```
- 解决AdaGrad学习率衰减问题

### 4.3 Adam
```
m_t = β1 * m_{t-1} + (1-β1) * ∇_θ J(θ)
v_t = β2 * v_{t-1} + (1-β2) * (∇_θ J(θ))²

m̂_t = m_t / (1 - β1^t)
v̂_t = v_t / (1 - β2^t)

θ = θ - η * m̂_t / (√v̂_t + ε)
```
- 默认: β1=0.9, β2=0.999, ε=1e-8
- 结合动量和自适应学习率

### 4.4 AdamW
```
θ = θ - η * (m̂_t / (√v̂_t + ε) + λ * θ)
```
- 解耦权重衰减
- 更好的泛化性能

## 5. 学习率调度

### 5.1 Step Decay
```
η_t = η_0 * γ^(floor(t / step))
```

### 5.2 Cosine Annealing
```
η_t = η_min + 0.5 * (η_max - η_min) * (1 + cos(πt/T))
```

### 5.3 Warmup
```
η_t = η_0 * t / T_warmup
```
训练初期使用较小学习率。

### 5.4 One-Cycle Policy
先增后减，配合动量相反变化。

## 6. 算法对比

| 算法     | 收敛速度 | 内存 | 超参数    | 推荐场景 |
|----------|----------|------|-----------|----------|
| SGD      | 慢       | 低   | η         | 简单任务 |
| Momentum | 中       | 中   | η,γ       | 通用     |
| Adam     | 快       | 中   | η,β1,β2   | 默认选择 |
| AdamW    | 快       | 中   | η,β1,β2,λ | 带正则化 |

## 7. 实践建议

1. **从Adam开始**: 学习率1e-3到3e-4
2. **微调时用SGD**: 配合小学习率和动量
3. **使用学习率调度**: Cosine或Step
4. **监控梯度**: 检查梯度爆炸/消失
5. **梯度裁剪**: 防止梯度爆炸
