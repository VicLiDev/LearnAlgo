# 卷积神经网络 (CNN)

## 1. 简介

卷积神经网络专门用于处理具有网格结构的数据，如图像。通过卷积操作提取局部特征。

## 2. 核心组件

### 2.1 卷积层
```
output[i,j] = Σ Σ input[i+m,j+n] * kernel[m,n]
```

**参数**:
- kernel_size: 卷积核大小
- stride: 步长
- padding: 填充

**输出尺寸**:
```
out_size = (in_size - kernel_size + 2*padding) / stride + 1
```

### 2.2 池化层
- **最大池化**: 取窗口内最大值
- **平均池化**: 取窗口内平均值
- 作用: 降维、增大感受野、防止过拟合

### 2.3 全连接层
将特征展平后进行分类。

## 3. 经典架构

### 3.1 LeNet-5 (1998)
- 结构: Conv-Pool-Conv-Pool-FC-FC
- 参数: ~60K
- 应用: 手写数字识别

### 3.2 VGG (2014)
- 特点: 使用3x3小卷积核堆叠
- VGG16: 13个卷积层 + 3个全连接层

### 3.3 ResNet (2015)
- 残差连接: `output = F(x) + x`
- 解决深层网络退化问题

### 3.4 Inception
- 多尺度卷积并行
- 1x1卷积降维

## 4. 关键技术

### 4.1 Batch Normalization
```
x_norm = (x - μ) / σ
out = γ * x_norm + β
```
加速训练，允许更大学习率。

### 4.2 Dropout
训练时随机丢弃神经元，防止过拟合。

### 4.3 数据增强
- 随机裁剪、翻转
- 颜色抖动
- Mixup/CutMix

## 5. 感受野计算

```
RF_l = RF_{l-1} + (kernel_size - 1) * ∏_{i=0}^{l-1} stride_i
```

## 6. 参数计算

|----------|-------------------|
| 层类型   | 参数量            |
| 卷积层   | C_in × C_out × K² |
| 全连接层 | N_in × N_out      |

## 7. 应用场景

- 图像分类
- 目标检测
- 语义分割
- 人脸识别
