# 环路滤波 (In-Loop Filtering)

## 1. 简介

环路滤波在解码环路内对重建图像进行滤波，减少块效应和振铃效应，提高图像质量和预测精度。

## 2. 去块滤波 (Deblocking Filter)

### 2.1 块效应原因

```
块边界处:
- 量化误差不一致
- 预测残差不连续
- 变换截断误差
```

### 2.2 滤波强度

| 条件                 | 强度 |
|----------------------|------|
| 边界两侧都是帧内预测 | 强   |
| 边界有非零系数       | 中   |
| 边界无系数           | 弱   |
| 运动矢量差大         | 弱   |

### 2.3 滤波决策

```
1. 计算边界两侧梯度
2. 根据QP和边界强度选择滤波器
3. 检查是否需要滤波 (避免过度模糊)
```

### 2.4 滤波操作

```
强滤波 (BS=4): 修改4个像素
  p1' p0' | q0' q1'

弱滤波 (BS<4): 修改2个像素
  p0' | q0'

滤波值 = (p0 + q0 + 1) >> 1  (简单平均)
或使用更复杂的插值
```

## 3. SAO (Sample Adaptive Offset)

### 3.1 基本思想

```
对每个像素加一个偏移值:
Rec'(x,y) = Rec(x,y) + Offset
```

### 3.2 SAO类型

| 类型 | 说明 |
|------|------|
| BO (Band Offset) | 按像素值分带 |
| EO (Edge Offset) | 按边缘方向分类 |

### 3.3 边缘偏移 (EO)

```
4种边缘模式:
  模式0: 水平    模式1: 垂直
  模式2: 135°    模式3: 45°

    a           a           b           c
  c p b       b p c       a p c       a p b
    b           a           b           a

根据p与a,b,c的关系确定类别和偏移
```

### 3.4 SAO参数编码

```
每个CTU编码:
- SAO类型 (无/BO/EO)
- 偏移值 (每个类别4位)
- 合并标志 (从相邻CTU合并参数)
```

## 4. ALF (Adaptive Loop Filter)

### 4.1 维纳滤波

```
目标: 最小化原始与重建的MSE

滤波器系数 w 通过训练得到:
Rec'(x,y) = Σ w[i,j] × Rec(x+i, y+j)
```

### 4.2 自适应特性

```
- 按CTU选择不同滤波器
- 根据局部特性调整
- 可选择是否启用ALF
```

### 4.3 滤波器形状

```
菱形滤波器:
    ○
  ○ ○ ○
○ ○ ● ○ ○
  ○ ○ ○
    ○

●: 中心像素
○: 滤波器系数
```

## 5. 滤波顺序

```
H.264:
  解码 → 去块滤波 → 存储

H.265:
  解码 → 去块滤波 → SAO → 存储

H.266:
  解码 → 去块滤波 → SAO → ALF → 存储
```

## 6. 性能影响

| 滤波器 | 复杂度 | BD-rate增益 |
|--------|--------|-------------|
| 去块滤波 | 5-10% | -5% ~ -10% |
| SAO | 2-5% | -2% ~ -5% |
| ALF | 10-20% | -5% ~ -8% |
