# 量化 (Quantization)

## 1. 简介

量化是视频压缩中唯一引入失真的环节，通过降低系数精度来减少数据量。

## 2. 标量量化

### 2.1 基本量化

```
Q(x) = round(x / QP)

反量化: x' = Q(x) × QP
```

### 2.2 量化误差

```
误差 = x - x' = x - round(x/QP) × QP

最大误差: ±QP/2
```

## 3. 变换量化一体化

### 3.1 H.264量化公式

```
正向量化:
|Zij| = (|Wij| × PF + f) >> qbits

反量化:
W'ij = Zij × V × 2^(qbits/6)

其中:
- PF: 预缩放因子 (取决于块位置)
- V: 反量化缩放因子
- qbits = 15 + floor(QP/6)
- f = 2^(qbits)/6 (帧内) 或 2^(qbits)/3 (帧间)
```

### 3.2 QP与量化步长

| QP | QStep | QP | QStep |
|----|-------|----| -------|
| 0  | 0.625 | 26 | 10.0 |
| 6  | 1.25  | 32 | 20.0 |
| 12 | 2.5   | 38 | 40.0 |
| 18 | 5.0   | 51 | 224.0 |

关系: QStep(QP+6) = 2 × QStep(QP)

## 4. 率失真优化量化 (RDOQ)

### 4.1 基本思想

```
传统量化: 独立量化每个系数
RDOQ: 联合考虑量化对率失真的影响
```

### 4.2 RDOQ流程

```
1. 对每个系数计算候选量化值 {q-1, q, q+1}
2. 使用动态规划找最优量化组合
3. 目标: min(D + λ×R)
```

### 4.3 计算复杂度

- 传统量化: O(N)
- RDOQ: O(N × candidates^N) → 需要优化

## 5. 量化矩阵

### 5.1 默认矩阵

```
用于调整不同频率系数的量化精度:

     DC  AC1 AC2 AC3 ...
DC  [16  16  16  16 ...]
AC1 [16  16  16  16 ...]
AC2 [16  16  16  16 ...]
...

高频通常使用更大的量化步长
```

### 5.2 自定义矩阵

- 可在SPS/PPS中定义
- 针对不同应用场景优化

## 6. 死区量化

### 6.1 原理

```
标准量化: [-QP/2, QP/2] 都量化为0

死区量化: 扩大零区间
[-QP×(1+β)/2, QP×(1+β)/2] → 0

β > 0 时，更多系数量化为0
```

### 6.2 效果

- 增加零系数数量
- 提高压缩效率
- 可能损失细节

## 7. 自适应量化

### 7.1 基于块的AQ

```
QP' = QP + ΔQP

ΔQP 取决于:
- 块复杂度
- 视觉敏感度
- 编码模式
```

### 7.2 基于宏块的AQ

```
高纹理区域: 降低QP (保留细节)
平坦区域: 增大QP (节省比特)
```
