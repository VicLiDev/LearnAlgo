# 预测编码 (Prediction)

## 1. 简介

预测编码利用已编码数据预测当前数据，只编码预测残差。残差能量比原始数据小很多，便于压缩。

## 2. 预测类型

| 类型             | 说明                  | 压缩效率 |
|------------------|-----------------------|----------|
| 帧内预测 (Intra) | 利用同一帧内相邻块    | 中等     |
| 帧间预测 (Inter) | 利用已编码帧 (参考帧) | 高       |

## 3. 帧内预测 (Intra Prediction)

### 3.1 H.264帧内预测模式

**4×4亮度块 - 9种模式:**

```
模式0 (垂直):          模式1 (水平):        模式2 (DC):
A B C D                A . . .              .
A B C D                B . . .              X (平均值)
A B C D                C . . .
A B C D                D . . .
```

```
模式3 (对角左下):      模式4 (对角右下):
D / / /               / / / A
  D / /             / / A .
    D /           / A . .
      D         A . . .
```

**16×16亮度块 - 4种模式:**
- 垂直 (模式0)
- 水平 (模式1)
- DC (模式2)
- 平面 (模式3)

**8×8色度块 - 4种模式:**
与16×16亮度相同

### 3.2 H.265帧内预测

- 支持4×4到32×32多种块大小
- 35种预测模式（33种角度 + DC + Planar）
- 更精细的角度预测

### 3.3 预测模式选择

使用率失真优化 (RDO) 选择最优模式：

```
Cost = D + λ * R

D: 失真 (SAD 或 SSD)
R: 编码比特数
λ: 拉格朗日乘子
```

## 4. 帧间预测 (Inter Prediction)

### 4.1 基本概念

```
当前帧 (P/B帧) ──→ 运动估计 ──→ 预测块
                      ↑
参考帧 ──────────────┘
```

### 4.2 预测类型

| 帧类型 | 参考帧    | 特点                 |
|--------|-----------|----------------------|
| I帧    | 无        | 独立编码，随机访问点 |
| P帧    | 前向参考  | 单向预测             |
| B帧    | 前向+后向 | 双向预测，压缩率最高 |

### 4.3 运动补偿

```
预测块 = 参考帧[y + mv_y, x + mv_x]

其中 (mv_x, mv_y) 是运动矢量
```

### 4.4 亚像素插值

**1/4像素精度 (H.264):**

```
整像素位置:     A
              |
半像素位置:   - B -
              |
1/4像素:     C A C
              D
```

插值滤波器：
- 半像素: 6抽头滤波器
- 1/4像素: 双线性插值

## 5. 高级帧间预测技术

### 5.1 多参考帧

```
参考帧列表: [Ref0, Ref1, ..., RefN-1]

P帧: 使用List0
B帧: 使用List0 + List1
```

### 5.2 加权预测

```
P(x,y) = w0 * Ref0(x+mv0_x, y+mv0_y) + w1 * Ref1(x+mv1_x, y+mv1_y)

B帧双向预测的加权平均
```

### 5.3 B帧的直接模式

- 利用时域/空域相邻块推导MV
- 跳过MV编码，节省比特

## 6. 预测残差

### 6.1 残差特性

```
残差 = 原始 - 预测

特点:
- 均值接近0
- 能量集中在低频
- 大部分值较小
```

### 6.2 残差编码流程

```
残差 → 变换(DCT) → 量化 → 熵编码 → 码流
```
