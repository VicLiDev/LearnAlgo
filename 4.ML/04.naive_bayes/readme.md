# 贝叶斯算法 (Bayesian Algorithms)

## 概述

贝叶斯算法是基于贝叶斯定理的一类算法，通过计算后验概率来进行分类或回归。其核心思想是利用先验知识和观测数据来更新对事件概率的估计。

## 贝叶斯定理

### 基本公式
```
P(A|B) = P(B|A) × P(A) / P(B)
```

- **P(A|B)**: 后验概率 (Posterior) - 在B发生的情况下A发生的概率
- **P(B|A)**: 似然 (Likelihood) - 在A发生的情况下B发生的概率
- **P(A)**: 先验概率 (Prior) - A发生的概率
- **P(B)**: 边缘概率 (Evidence) - B发生的概率

### 用于分类
```
P(y|x) = P(x|y) × P(y) / P(x)
```

选择使后验概率最大的类别:
```
ŷ = argmax_y P(y) × P(x|y)
```

## 常用贝叶斯分类器

### 1. 朴素贝叶斯 (Naive Bayes)

#### 特点
- 假设特征之间相互独立（朴素假设）
- 计算简单，效率高
- 对小规模数据表现良好

#### 公式
```
P(y|x₁,x₂,...,xₙ) ∝ P(y) × ∏P(xᵢ|y)
```

#### 类型

##### 高斯朴素贝叶斯 (Gaussian Naive Bayes)
- **假设**: 特征服从高斯分布
- **公式**: P(xᵢ|y) = (1/√(2πσ²)) × exp(-(xᵢ-μ)²/(2σ²))
- **适用**: 连续型特征

##### 多项式朴素贝叶斯 (Multinomial Naive Bayes)
- **假设**: 特征服从多项分布
- **适用**: 文本分类、词频统计

##### 伯努利朴素贝叶斯 (Bernoulli Naive Bayes)
- **假设**: 特征服从伯努利分布（二值）
- **适用**: 二值特征、文档分类

##### 补充朴素贝叶斯 (Complement Naive Bayes)
- **特点**: 对不平衡数据的改进
- **适用**: 不平衡数据集

### 2. 贝叶斯网络 (Bayesian Network)

- 表示变量之间的依赖关系
- 使用有向无环图 (DAG)
- 每个节点表示一个变量
- 边表示依赖关系

### 3. 高斯过程 (Gaussian Process)

- 非参数贝叶斯方法
- 用于回归和分类
- 可以给出预测的不确定性

## 优缺点

### 优点

1. **计算效率高**
   - 训练和预测速度快
   - 适合大规模数据

2. **小样本表现好**
   - 在数据量较少时也能工作
   - 不需要大量训练数据

3. **处理高维数据**
   - 适合文本分类等高维问题
   - 特征维度高时仍然高效

4. **可解释性强**
   - 基于概率的输出
   - 容易理解

5. **增量学习**
   - 可以在线更新模型
   - 适合流数据

6. **对缺失值不敏感**
   - 可以自然处理缺失特征

### 缺点

1. **独立性假设**
   - 朴素贝叶斯假设特征独立
   - 实际中特征往往相关

2. **零概率问题**
   - 某些特征组合可能不存在
   - 需要平滑技术（拉普拉斯平滑）

3. **对输入数据的形式敏感**
   - 需要选择合适的分布假设
   - 连续特征需要假设分布

## 拉普拉斯平滑

解决零概率问题:
```
P(xᵢ|y) = (count(xᵢ,y) + α) / (count(y) + α × n)
```

- α: 平滑参数（通常为1）
- n: 可能的取值数量

## 使用建议

### 1. 选择合适的类型
- 连续特征: 高斯朴素贝叶斯
- 文本分类: 多项式朴素贝叶斯
- 二值特征: 伯努利朴素贝叶斯

### 2. 数据预处理
- 标准化（对高斯朴素贝叶斯）
- 特征选择
- 处理缺失值

### 3. 参数调优
- alpha (平滑参数)
- fit_prior (是否学习先验)

### 4. 处理特征相关性
- 如果特征高度相关，考虑其他算法
- 或使用PCA降维
