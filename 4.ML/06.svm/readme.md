# 支持向量机算法 (Support Vector Machine, SVM)

## 概述

支持向量机是一种强大的监督学习算法，可用于分类、回归和异常检测。SVM的核心思想是找到一个最优的超平面，将不同类别的数据分开，同时最大化分类间隔。

## 基本概念

### 1. 超平面
在n维空间中，超平面是n-1维的子空间：
- 2D空间：直线
- 3D空间：平面
- 高维空间：超平面

### 2. 支持向量
距离决策边界最近的训练样本点，这些点决定了超平面的位置。

### 3. 间隔 (Margin)
超平面到最近数据点的距离。SVM试图最大化这个间隔。

### 4. 线性可分
数据可以用一个超平面完全分开。

## 数学原理

### 1. 线性SVM

#### 硬间隔（线性可分）
```
最小化: (1/2)||w||²
约束条件: yᵢ(w·xᵢ + b) ≥ 1
```

#### 软间隔（允许一些错误）
```
最小化: (1/2)||w||² + C Σξᵢ
约束条件: yᵢ(w·xᵢ + b) ≥ 1 - ξᵢ
          ξᵢ ≥ 0
```

### 2. 对偶问题
通过拉格朗日乘子法，可以将原问题转化为对偶问题：
```
最大化: Σαᵢ - (1/2)ΣΣαᵢαⱼyᵢyⱼ(xᵢ·xⱼ)
约束条件: Σαᵢyᵢ = 0, 0 ≤ αᵢ ≤ C
```

### 3. 核函数 (Kernel Trick)
将数据映射到高维空间，使其在高维空间线性可分：
```
K(xᵢ, xⱼ) = φ(xᵢ)·φ(xⱼ)
```

## 常用核函数

### 1. 线性核 (Linear Kernel)
```
K(x, y) = x·y
```
- 适用：线性可分数据
- 优点：速度快，不易过拟合
- 缺点：只能处理线性关系

### 2. 多项式核 (Polynomial Kernel)
```
K(x, y) = (γ·x·y + r)^d
```
- 参数：γ (gamma), r (coef0), d (degree)
- 适用：非线性关系
- 优点：可以拟合复杂的非线性边界

### 3. 径向基函数核 (RBF Kernel / Gaussian Kernel)
```
K(x, y) = exp(-γ||x - y||²)
```
- 参数：γ (gamma)
- 适用：大多数非线性问题
- 优点：灵活，性能好
- 缺点：容易过拟合，需要调参

### 4. Sigmoid核
```
K(x, y) = tanh(γ·x·y + r)
```
- 类似神经网络的激活函数

## 核函数选择

1. **样本数少，特征多**：线性核
2. **样本数和特征数都不大**：RBF核
3. **样本数大**：线性核或多项式核
4. **图像数据**：线性核或RBF核

## 超参数

### C (正则化参数)
- 控制对误分类的惩罚程度
- **C大**：严格分类，可能过拟合
- **C小**：允许更多错误，可能欠拟合

### gamma (RBF核参数)
- 控制单个样本的影响范围
- **gamma大**：影响范围小，决策边界复杂，可能过拟合
- **gamma小**：影响范围大，决策边界平滑，可能欠拟合

### degree (多项式核)
- 多项式的次数
- 通常取2或3

### coef0
- 多项式和sigmoid核的独立项

## 优缺点

### 优点
1. **高维空间有效**：在特征维度很高时表现良好
2. **内存效率**：只使用支持向量
3. **泛化能力强**：通过最大化间隔提高泛化能力
4. **核函数灵活**：可以处理非线性问题
5. **理论完善**：有坚实的统计学习理论基础

### 缺点
1. **大数据集慢**：训练时间复杂度O(n²)到O(n³)
2. **参数敏感**：需要仔细调参
3. **特征缩放敏感**：需要标准化或归一化
4. **黑盒模型**：难以解释
5. **噪声敏感**：对噪声数据敏感

## SVM回归 (SVR)

与分类类似，但目标是找到一个超平面，使大部分点落在ε-带内：
```
最小化: (1/2)||w||² + C Σ(ξᵢ + ξᵢ*)
约束条件: -ε - ξᵢ* ≤ yᵢ - (w·xᵢ + b) ≤ ε + ξᵢ
```

## 使用建议

### 1. 数据预处理
- **特征缩放**：必须标准化或归一化
- **处理异常值**：SVM对异常值敏感
- **处理缺失值**：SVM不能直接处理缺失值

### 2. 参数调优
- 使用网格搜索 (GridSearchCV)
- 优先调整 C 和 gamma
- 使用交叉验证评估

### 3. 核函数选择
- 先尝试线性核
- 效果不好再尝试RBF核
- 根据数据特点选择合适的核

### 4. 大数据集
- 使用LinearSVC（线性核的优化实现）
- 考虑使用SGDClassifier
- 减少训练样本量

## 应用场景

- 文本分类
- 图像识别
- 手写识别
- 生物信息学
- 金融预测
- 医疗诊断

## 与其他算法对比

| 特性 | SVM | 随机森林 | 神经网络 | 逻辑回归 |
|------|-----|----------|----------|----------|
| 小样本 | 好 | 好 | 差 | 好 |
| 大样本 | 慢 | 好 | 好 | 好 |
| 高维数据 | 好 | 好 | 差 | 好 |
| 非线性 | 好 | 好 | 好 | 差 |
| 可解释性 | 低 | 中 | 低 | 高 |
| 调参难度 | 高 | 低 | 高 | 低 |
