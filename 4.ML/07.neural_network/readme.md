# 神经网络算法 (Neural Networks)

## 概述

神经网络是一种模拟人脑神经元连接方式的机器学习算法，由多层节点（神经元）组成，
通过调整连接权重来学习数据的复杂模式。

## 基本概念

### 1. 神经元 (Neuron)
神经网络的基本单元，接收输入，进行加权求和，经过激活函数后输出：
```
y = f(Σwᵢxᵢ + b)
```
- wᵢ: 权重
- xᵢ: 输入
- b: 偏置
- f: 激活函数

### 2. 层 (Layer)
- **输入层**: 接收原始特征
- **隐藏层**: 进行特征变换和学习
- **输出层**: 输出预测结果

### 3. 前向传播 (Forward Propagation)
数据从输入层经过隐藏层到达输出层的过程。

### 4. 反向传播 (Backpropagation)
根据损失函数，通过链式法则计算梯度，从输出层向输入层更新权重。

## 激活函数

### 1. Sigmoid
```
σ(x) = 1 / (1 + e⁻ˣ)
```
- 范围: (0, 1)
- 优点: 平滑，输出可解释为概率
- 缺点: 梯度消失，输出非零中心

### 2. Tanh
```
tanh(x) = (eˣ - e⁻ˣ) / (eˣ + e⁻ˣ)
```
- 范围: (-1, 1)
- 优点: 零中心
- 缺点: 梯度消失

### 3. ReLU (Rectified Linear Unit)
```
ReLU(x) = max(0, x)
```
- 范围: [0, +∞)
- 优点: 计算快，缓解梯度消失
- 缺点: 死亡ReLU问题

### 4. Leaky ReLU
```
LeakyReLU(x) = max(αx, x)
```
- 解决死亡ReLU问题
- α通常取0.01

### 5. Softmax
```
softmax(xᵢ) = eˣⁱ / Σeˣʲ
```
- 用于多分类输出层
- 输出和为1

## 损失函数

### 1. 均方误差 (MSE)
```
L = (1/n) Σ(y - ŷ)²
```
- 用于回归

### 2. 交叉熵 (Cross-Entropy)
```
L = -Σy log(ŷ)
```
- 用于分类

## 优化算法

### 1. 随机梯度下降 (SGD)
```
w = w - η∇L
```
- 简单但可能收敛慢

### 2. Momentum
```
v = γv + η∇L
w = w - v
```
- 加速收敛

### 3. Adam
- 自适应学习率
- 结合Momentum和RMSprop
- 最常用的优化器

## MLP (多层感知机)

### 结构
- 全连接的前馈神经网络
- 多个隐藏层
- 非线性激活函数

### 特点
- 可以学习非线性关系
- 通用近似定理：可以近似任何连续函数

## 优缺点

### 优点
1. **强大的表达能力**: 可以学习复杂的非线性关系
2. **通用近似**: 理论上可以逼近任何函数
3. **自动特征提取**: 不需要手动特征工程
4. **可扩展性强**: 可以堆叠多层

### 缺点
1. **黑盒模型**: 难以解释
2. **需要大量数据**: 小数据集容易过拟合
3. **调参复杂**: 需要调整很多超参数
4. **计算资源**: 训练可能需要大量计算
5. **训练不稳定**: 可能陷入局部最优

## 超参数

### 1. 网络结构
- **hidden_layer_sizes**: 隐藏层大小，如(100, 50)表示两个隐藏层
- **层数**: 隐藏层数量
- **每层神经元数**: 每层的节点数

### 2. 训练参数
- **learning_rate_init**: 初始学习率
- **max_iter**: 最大迭代次数
- **batch_size**: 批量大小
- **alpha**: L2正则化参数

### 3. 激活函数
- 'relu': ReLU（默认）
- 'tanh': Tanh
- 'logistic': Sigmoid
- 'identity': 无激活（线性）

### 4. 优化器
- 'adam': Adam（默认）
- 'sgd': 随机梯度下降
- 'lbfgs': 拟牛顿法

## 使用建议

### 1. 数据预处理
- **标准化**: 神经网络对特征缩放敏感
- **归一化**: 将数据缩放到[0,1]或[-1,1]
- **处理缺失值**: 神经网络不能处理缺失值

### 2. 网络结构选择
- 从简单结构开始
- 逐步增加复杂度
- 使用交叉验证评估

### 3. 防止过拟合
- **正则化**: 使用L2正则化
- **Dropout**: 随机丢弃神经元
- **早停**: 监控验证集性能
- **减少网络大小**: 减少层数或神经元数

### 4. 参数调优
- 先调整学习率
- 然后调整网络结构
- 最后调整正则化参数
